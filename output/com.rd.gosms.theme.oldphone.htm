Go SMS Pro Theme 1998

Updated for better compatibility with Go SMS Pro 5.0+

This is an minimal retro 8-bit black and green theme for Go SMS Pro, based on the infamous Nokia phones of the late 90s. Simple and nostalgic.

Blue version also available
Check out the matching Go Keyboard, Go Contacts EX and Go Launcher themes

*PLEASE READ THE INSTRUCTIONS AND FAQs*

*JELLY BEAN INFO*
Font may not show up when scanned. Install ER Designs Fonts Pack and scan again.

**Fonts INCLUDED**

***This Go SMS Pro Theme only works with GO SMS Pro.
***Install GO SMS Pro free from the market first to use this Go SMS Theme.


FREQUENTLY ASKED QUESTIONS

--The font isn't showing up after I scan for font packages--
Install my font packages app, called ER Designs Fonts Pack.

--How do I apply the theme?--
Please do not open directly after it's installed as you would a normal application. Go to the GO SMS main screen (inbox): MENU > Theme. Make sure you have "Installed" selected, not "Featured". Choose and apply the theme.
Under Settings -> Application settings -> Running mode, make sure you have appearance customization enabled.

--How do I apply the included custom font(s)?--
The font will not apply itself automatically.
To apply fonts go to Appearance Settings (Menu -> Settings -> Appearance Settings) and press "Scan font packages" then apply in customization.

--Why doesn't the theme look right?--
Theme looks best with GO CHAT DISABLED.
Remember to set chat style to BUBBLE, in both Conversation customization and Conversation other settings (both are in Appearance Settings). You may need to uncheck and recheck them even if they are already selected.

--How do I disable Go Chat?--
Menu -> Settings -> Application Settings -> Running Mode and chose an option with GO Chat Disabled

--Why isn't the custom Pop Up showing?--
Make sure "Enable popup windows" is checked in Notification Settings under menu.
From the inbox, scroll left to the "Theme Store" icon and press it. Select "Go Popup" from the top menu, press the settings icon in the top right and select "Use the normal popup".

--What is this theme for?--
This theme only works for Go SMS Pro. It will not theme anything else on your phone.

Any other questions, concerns, suggestions, please email me directly at erikryerson@gmail.com

常见问题

- 字体显示不出来后，我扫描字体包 -
我的字体安装包的应用程序，称为ER设计字体包。

- 我该如何申请的主题 -
请不要直接打开安装成功后，你将一个正常的应用程序。 GO短信主屏（收件箱）：菜单>主题。请确保您已选择“安装”，而不是“精选”。选择和应用的主题。
在“设置” - >“应用程序设置” - >运行模式，确保你有启用外观定制。

- 我该如何申请包括自定义字体（S） -
字体不会自动应用。
要套用字体外观设置（菜单 - >设置 - >外观设置），然后按“扫描字体包”适用于定制。

- 为什么不主题看起来吧？ -
主题看起来最好GO CHAT禁用。
请记住，在两个对话的定制和对话的其他设置（无论在外观设置）的设置聊天风格泡。您可能需要取消选中并重新检查它们，即使它们已被选中。

- 我如何禁用聊天？ -
菜单 - >设置 - >应用程序设置“ - >”运行模式，并选择一个选项与GO聊天残疾人

- 为什么没有自定义弹出显示 -
确保“启用弹出窗口”被选中在“通知设置”菜单下。
从收件箱，请向左滚动“主题商店”图标，然后按。从顶部的菜单中选择“弹出”，在右上角按“设置”图标，并选择“使用正常的弹出窗口”。

- 这是什么主题？ -
这个主题只适用于GO短信临。它不会在您的手机主题别的。

任何其他问题，关注，建议，请给我发电子邮件，直接在erikryerson@gmail.com
