If you have any trouble using spotimote, please send me an e-mail or leave a message on the forum (http://www.evolu-on.com/forum).

Spotimote communicates with the Spotimote server component over Wifi. Make sure your phone and your computer are connected on the same wifi network to establish the connection.

After the installation of the Spotimote server component, when you launch it for the first time, you may be prompted by the Windows Firewall. You have to authorize the server component to access the network or it won't be able to communicate with your phone/tablet.

About the permissions, the "READ PHONE STATUS AND IDENTITY" permission is required to let spotimote automatically pause the music when an incoming call is received on the phone.
