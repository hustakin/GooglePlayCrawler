------------------------------------------------------------------------------------------------------------------------

스마터 서브웨이(Smarter Subway)는 기존의 획일적인 지하철어플에서 벗어나 사용자입장에서 보다 정확하고 편리한, 그리고 더욱 똑똑한 지하철 노선도 앱을 목표로 하고 있습니다.

수도권/부산/대구/대전/광주 등 전국 모든 지하철 노선도가 추가되었습니다

기존 지하철 앱의 내용을 기반으로 다양한 기능을 추가/강화하여 Smarter Subway와 함께 더욱 편리하게 지하철을 이용하 실 수 있습니다.

기존 앱은 일률적인 소요시간 계산방법으로 정확한 소요시간을 도출하기 힘들었고, 대략 몇 분이 걸리는지만 알려 줄 뿐, 사용자가 언제 열차를 타면 원하는 시간에 도착할 수 있는지를 제시하지 않았습니다.

환승시간이 언제인지를 알려주고 열차가 금방 있을 경우 서두르면 잡을 수 있을지 또는 없을지를 알려주는, 배차간격이 짧은 출퇴근 시간대와 열차간격이 긴 막차시간대 등을 고려한 사용자 편의를 중심으로 한 지하철 어플리케이션 Smarter Subway를 소개합니다.

1. 다양한 경로 검색 방식
Smarter Subway는 앱이 이동 경로를 검색하는 방식을 시간에 따라서 [출발시간기준 / 도착시간기준 / 막차검색]과경로에 따라서 [최소시간 / 최단거리 / 최소환승] 으로 제공합니다.


2. 동적인 결과창
Smarter Subway는 정적인 결과 창이 아닌, 결과를 원할 경우 수시로 변경할 수 있는 동적인 결과창을 도입하여 편리성을 증대시켰습니다.
결과창에서 출발역과 도착역 / 요일 / 시간등을 다시 선택할 수 있으며, 검색된 경로의 이전/다음 열차를 확인 할 수 있습니다.


3. 빠른검색
역검색과 경로 검색을 합쳐서 빠르게 경로를 찾을 수 있는 기능입니다.
등록하신 "자주가는역"을 바로 출발역/도착역으로 설정 할 수 있어 편리함을 더 하였습니다.
("자주가는 역"은 해당 역을 터치 후 나타나는 팝업 메뉴의 정 가운데의 별 모양 버튼을 터치하므로써 추가/삭제할 수 있습니다.)

4. 최근검색
최근에 검색한 결과를 다시 볼 수 있습니다.
환승 시간을 알아보기 위해 이전에 검색하였던 내용을 다시 찾을 때 편리합니다.

5. 역검색
노선도에서 찾고자 하는 역의 위치를 모를 경우,역의 이름으로 해당역을 찾을 수 있는 기능입니다.
우측에 노선을 터치하므로써 노선 별로 검색이 가능합니다.

6. 주변역/현위치
현재의 GPS 값을 기반으로 하여, 주변에 가장 인접한 역이 무엇인지를 보여주는 "주변역" 기능과, 사용자 주변의 지도를 보여주는 "현위치" 기능으로 익숙치 않은 장소에서 보다 편리하게 역을 찾아 갈 수 있습니다.

1. 왜 스케줄이 맞지 않나요?
- 수도권의 경우 특히 1호선은 운행구간이 길어 거의 시간표와 동일하게 운행되지 않습니다. 그렇기 때문에 도착정보가 틀리게 됩니다. 실시간 정보의 경우는 서울시(모바일서울)에서 제공하고 있습니다만 실시간 정보 역시 도착정보가 틀릴 때가 있는데요 이는 열차를 운영하는 코레일, 서울메트로에서 열차 운행정보를 제공하지 않기 때문입니다.

- 스케줄을 확인 하시면 역에 붙어있는 스케줄과 동일할 것입니다. 항상 최신의 스케줄을 유지 할 수 있도록 노력하겟습니다.

2. 왜 우리지역은 나오지 않나요?
- 어플을 최초 설치하시면 지역은 기본으로 수도권으로 정해져 있습니다. 지방에서 이용하시는 분들은 메뉴키->설정에서 사용하시는 지역을 설정하시면 이용 가능하십니다.

스마트 지하철은 안드로이드와 iOS 모두 지원하며,
지하철 노선도에서 출발지와 도착지만 눌러주면 쉽고 빠르게 경로와 소요시간을 확인 할 수 있습니다.
또한, 지금 출발하면 얼마나 걸릴지, 막차기준 뿐만아니라, 원하는 시간을 지정하면 최소한 언제 출발해야 할지도 확인이 가능합니다.?


■ 지하철 노선도 ■
- 서울(수도권)/부산/대구/대전/광주 지역의 노선도 제공
- 수도권의 경우 최초로 의정부경전철과 수인선도 제공!!
- ZOOM IN/OUT 가능
- 노선도를 이중 탭시 확대가 되면서 그 위치가 중앙으로 이동되며 다시 이중 탭하면 노선도가 축소됨


■ 역명으로 검색하기 ■
- 일반 검색 및 초성 검색 기능 제공
- 역명 검색 결과를 실시간으로 제공


■ 역 상세정보 ■
=> 역을 탭하면 열차 도착 정보 박스 노출되며, 기본적인 도착현황과 기타 정보를 바로 확인 가능함
=> 사당/교대/서울역 같은 환승역의 경우, 역 선택 후 노출된 상단 박스 영역을 좌/우로 Flicking하면 호선별로 확인 가능함.

① 도착현황
- 선택한 역을 기준으로 양 방향 열차의 도착 정보 제공

② 열차시간표
: 선택한 역의 평일/토요일/공휴일의 시간표 제공
: 급행인 경우 노란색으로 표시~
: 환승 구간의 경우 각 호선 별로 열차 시간표 확인 가능

③ 출구정보
: 선택한 역의 출구정보 제공

④ 기타 정보
: 화장실 내/외부 위치
: 열리는 문의 방향
: 횡단 가능 여부


■ 경로 조회 방법 ■

① 출발/도착역 선택
: 해당 역을 최초 한번 터치하면 출발역으로 선택되고, 이어서 다른 역을 선택하면 도착역으로 지정됨
: 출발/도착역 핀을 길게 누르면 우측 상단에 휴지통 아이콘이 보이며, 그곳으로 끌어다 놓으면 출발 또는 도착역 선택이 취소됨

② 경유지 선택
: 출발→도착 중간에 다른곳을 들렀다가 가야 할 경우 또는 기본으로 제공되는 경로가 마음에 들지 않을 경우 환승 핀을 길게 눌러서 들릴 곳에 위치 시키면 됩니다.

③ 소요시간 확인(일반/급행 확인가능)
: 경로 조회 박스 = 전체소요시간/도착예정시간/정차역수/환승횟수 확인 가능
: 일반/급행
- 급행이 존재하는 경로인 경우 우측 검색 버튼 하단에 노출되는 일반 버튼을 터치하면 급행을 탄 경우 소요시간 및 경로 확인 가능함
- 버튼명이 급행이라고 보이면, 현재 보고 있는 정보가 급행에 대한 정보임
: 최소시간(이동경로의 최소시간 경로)와 최소환승(이동경로의 최소환승 경로) 조회 가능
: 소요시간은 환승통로 도보시간을 반영한 시간
※ 환승통로 도보시간 및 지하철 대기시간 관련
지하철 노선도위에서 보여주는 출발/도착 경로 상에서의 도보시간은 다른 앱과 비교시 도보시간을 조금 넉넉히 잡음

④ 경로 상세 정보 보기
: 경로조회 박스 선택 시 상세 정보 확인 가능함.(소요시간/정차역/내리는문/빠른환승 등)
: 출발 or 도착 시간을 변경하면,지정한 시간으로 소요 시간을 다시 계산 가능함.
: 도착했는데, 열차가 가버렸다면 다음열차 버튼을 한번 눌러주기만 하면 됨!
: 즐겨찾기(★) 담기 기능 제공
: 막차 버튼을 누르면 막차 시간으로 소요 시간 계산
- 일반 상세인 경우 또는 급행 상세인 경우 해당 하는 기준으로 소요 시간 계산됨
※ 경로 조회 박스를 눌러 경로 상세 정보에서 시간 정보를 수정 후 지하철 노선도로 다시 나온 경우 변경한 정보는 반영 안됨

⑤ 경로 수정
: 경로 선택 후 출발/도착/경우/환승 아이콘을 길게 누르면 위치 이동 가능
