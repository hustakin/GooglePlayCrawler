说明
★Description:
The ICS theme for GOWidget! Get the looks and feel of Android 4.0: Ice Cream Sandwich!

★Features:
This theme supports the Message (3D!), Email, Calendar, Default Search, Search, Contact, Bookmark, Switch, Facebook, Store, Note, Twitter and TaskManager widget.

★Perfect feel:
Use this theme in combination with ICS GO Launcher EX Theme made by me (Kani) to have the perfect Android 4.0 feel!

★How to use:
Please do not open this app after installation. You need to apply it in GO Launcher. Follow the instructions below.

1) Make sure you have installed GO Launcher EX (free app)
2) Make sure you have installed the widgets you would liek to use
3) Press and hold at the empty homescreen
4) Select GOWidget (bottom of screen)
5) You’ll see a list of the GOWidgets
6) Select the GOWidget you would like to use, for example choose Message
7) You’ll see a preview of the widget now and two buttons at the bottom
8) Press the left button which says “Default”
9) A list will pop-up with all the themes that you have installed
10) Choose ICS
11) You’ll see a preview of ICS Message now
12) Press the “Add to screen” button
13) You will be directed to a black screen with several options
14) After you are done with those setttings, just press OK
15) The GOWidget in ICS theme is added to your homescreen now

If you need help with this theme, please e-mail to info@mifavo.com.

★Rate this app:
Your feedback is important to me in order to improve this theme. Please do not rate this theme 1 star because you can't "open" the theme or can't find the theme. Please e-mail me and give me a chance to help you solve the problem.

★Contact:
Please feel free to e-mail me for suggestions and questions. Let's make this theme as close to ICS as possible!

★More info:
Please visit www.mifavo.com to read more about future updates and new themes.
