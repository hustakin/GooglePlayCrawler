说明
*** PLEASE PLEASE PLEASE check out the demo version first!!! ***

This is Almost Smart Cover.

This application uses proximity sensor to turn on/off your device's screen. Meaning, if you have a sort of cover, it will be like iPad2's smart cover.

Main Feature:
1. Uses phone's cover (if available) to turn off and on screen.
2. Just putting the phone into pocket will turn off the screen, and vice versa.
3. Provides simple widget to enable/disable the feature temporarily.
4. Can set turn off delay time to prevent unwanted screen off.

* NEW FEATURE
1. Option to show/hide notification when started.
2. Auto start when booted.
3. Vibrate when screen is turned off.
4. Register disable application. When these application is running, the sensor will be disabled.
** All of the above features will be configured in Settings **

尋找Android版本的smart cover嗎?
現在，你可以使用普通的保護蓋當作可以開關螢幕的保護蓋。
或是也可以直接使用手指接觸接近感測器以開關螢幕。

功能
1.自動開啟螢幕
2.自動關閉螢幕
3.設定例外程式
4.螢幕橫向時自動停止功能，避免誤觸
5.自動解鎖，部分Android版本可能不適用
6.螢幕關閉前顯示提示
7.藉由提示列或捷徑暫停啟動自動關閉螢幕
8.藉由widget關閉螢幕
9.睡眠時自動停止功能，節約電量
10.使用連續觸碰啟動螢幕。
11.通話自動暫停程式

注意：
1.當系統記憶體不夠時，可能會把此軟體關閉，故有時需要自行重新啟動。
2.離開設定時請使用返回鍵，不要用home鍵，不然有可能在開完螢幕後再次出現。
3.反安裝前務必要先取消螢幕鎖定權限。
4.在Android4.1.2跟4.2之後的版本，當螢幕開啟後超過設定的"解鎖時限"時，會強制關閉螢幕，而非像以前會因為解鎖到一半而不關閉。這是為了修正4.1.2之後開啟螢幕後不會自動關閉螢幕的問題。

如果只是想要自動開啟螢幕，可以使用我的另一軟體"Auto screen on"，他有更多開發型選項，另外還可以藉由重力感測器開啟螢幕，並且提供記錄模式，讓您確認功能是否正常執行。

本軟體專對手機設計，因為大部分手機才有接近感測器，只有少數平板有接近感測器。

如有功能建議歡迎寄信到sdyy.hsu@gmail.com

已知問題：
1.部分機器的接近感測器對於深色保護殼感應不正常（ Galaxy Note II ）導致功能可能不正確。
2.在強光下也可能導致接近感測器錯誤。
