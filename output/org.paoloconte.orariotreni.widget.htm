This app provides information about Chennai suburban/local train services running between two train stations at or after a given time of day.

You can search train services between any two stations which fall on the same direct route either on Weekdays or Sundays.

The search provides you with a list of trains with boarding time at source station and alighting time at destination stations.

You can press on any listed train to get full information about that specific train, like its start/in between/destination stations and time for each.

Among other features:
- You have an option to search nearby source/destination station based on phones last known Geo-coordinates (latitude/longitude)
- An option to long press on any station name to see it on a Google maps
- Look at the various routes on which suburban trains operate in Chennai
- Swap Source-Destination stations so as to find return trains easily
- Type Source-Destination station names
- The last searched station pair gets saved for quick use in future

NOTE - This app is not designed to be used from SD cards. Please install and keep it in the device memory itself.

This app is available for FREE. This app can work completely offline.

If an internet connection is available it would be used to only display advertisements.
