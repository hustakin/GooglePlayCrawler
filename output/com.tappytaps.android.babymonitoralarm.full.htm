Babyphone App mit einer sehr einfachen Benutzeroberfläche. Alle erforderlichen Parameter lassen sich individuell einstellen. Es gibt also kein "funktioniert nicht", sondern nur ein "schlecht eingestellt".

Da im Gegensatz zu einem echten Babyfon ein Anruf hergestellt wird muss die App unterscheiden, ob das Kind nur kurz unruhig ist und weiterschläft, oder schon tatsächlich wach ist. Daher wird eine gewisse Anzahl an Sekunden überwacht (default 15s) und ermittelt wie oft die Lautstärke (default auf mittlere Lautstärke wie lautes Reden) zu hoch war. Wird die Lautstärke in diesem Zeitraum mehrmals überschritten (default 4 mal) wird der Alarm ausgelöst.

Da Sie diese drei Werte anpassen können, sind Sie als Nutzer in der Lage das Verhalten des Babyfons ganz auf Ihre Bedürfnisse anzupassen.

Dieses Babyphone-App kann eine SMS senden, oder ein anderes Telefon anrufen (oder beides). Es können die Lautsprecher für den Aufruf aktiviert werden, sodass Sie auch zu Ihrem Baby sprechen können.

Sobald die App aktiv geschaltet wird, werden alle Telefonfunktionen stumm geschalten (Wecker, Anrufe, Nachrichten etc.). Die vorherigen Einstellungen werden automatisch reaktiviert, sobald die App geschlossen oder der Baby-Alarm ausgelöst wird.

How it works:
Noise is meassure using the internal microphone. Values are miltiplied by a "sensitivity" factor. Result is displayed as noise input "value". Whenever this noise value exceeds half of displayed width it is added to an "alarm level value".

Whenever "alarm value" exceeds a limit (which is called "threshold", BabyMonitor calls the specified phone number.
After a call ends, BabyMonitor sleeps for 0-10 minutes before it again starts this processing noise.
