On Board Diagnostic Code Reference contains well over 14,000 OBD I & OBD II trouble codes covering 50 different models of automobiles. Most models contain OBD I codes and all models support generic OBD II codes. This app does not connect to Bluetooth codes readers but is a great companion for code readers that do not contain internal databases. If the code you are looking for is not contained within our database then you can try searching the internet with the built in button. This app is compatible with all tablets and also features app to SD Card.
