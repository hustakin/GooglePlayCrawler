NOTE AND PROBLEMS:

Group message:

You should enable the group message support first: MultiMedia Mms settings -> Group messaging.

- 1. enable the "Block mms notification"
- 2. enable the "Group messaging".

Doesn't work:

- Some device can not work properly, if you meet similar problem, please contact with me first. I will help you!

Out order of message problem:

- If you do not want to see duplicate notifications, you can try to select "Block Sms Notification" option in settings, but this will cause two duplicate messages on some devices.
- If your messages is out of order, please try to select "Block Notification" option in settings of iphone message.

Contact problem:

- Please write email to barleystudio(at)gmail.com to report your question, I will be resolve you question quickly.

Emoji problem:

- Please press the menu and select "Insert Smile" to insert emoji.
- The iPhone user's IOS version must be > 5.01
- Because the emoji message is block by network carrier when across network carriers, So you can try to send or ask your friend to do like the follow:
- Android user:
- Press menu key and select "Add Subject", fill it with like "Emoji Subject" or any other title, then input the emojis.
- iPhone user:
- System Setting -> Messages -> "Show Subject Field"
- Tell your friend to send an emoji message with an subject.

MMS picture problem:

- If you can't send or receive mms, you can select "Multimedia message (MMS) settings" option in settings of iphone message, then select the "Fill APN info Manually", you can click MMSC/PROXY/PORT to see the value, if you can not see anything, you can enter your own configure, you can see this in follow path:
- android 2.3: system settings -> wireless & netowrks -> Mobile networks -> Access Point Names.
- android 4.0 : system settings -> Wireless network More ->  Mobile networks -> Access Point Names.
