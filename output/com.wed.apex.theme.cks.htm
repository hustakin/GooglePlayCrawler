This theme fully supports Apex Launcher, GO Launcher EX / HD, Nova Launcher, ADWLauncher EX / ADW.Launcher, Next Launcher 3D, Action Launcher Pro, TSF Shell*, Atom Launcher*, Holo Launcher HD*, Smart Launcher* GSLTHEME, SS Launcher, and more!

*You must manually apply theme on these launchers, full instructions on support website.

To change wallpaper:
Press Menu > Wallpaper > Scroll to Ultimate iOS7 Wallpapers > Set wallpapers

To change dock bar (Holo-Apple) icons:
First, choose whichever app you would like to use and drag to homescreen.
Long-press on that app's icon and select Edit > press the Icon > (Swipe left) and choose Ultimate iOS7 icon pack > Select a transparent icon to use. Then just drag it onto your dock bar.

To switch homescreen icons:
Long-press on that app's icon > Edit > Press the icon > Choose Ultimate iOS7 icon pack > Select an alternate icon style!
