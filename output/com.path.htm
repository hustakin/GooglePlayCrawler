说明
对于您的私生活，Path 是一款可靠的应用程序。使用这款应用程序时可以回归自然，与最重要的人保持紧密联系，分享生活而不是链接。通过 Path 可完全控制分享的生活瞬间并随意选择分享对象。Path 设计精美，易于使用，一直拥有 5 星评级，受到数百万人的喜爱，而且没有广告。
-“简而言之，我家人的生活因为 Path 而变得更好。无论我们相距多么遥远，看到他们‘看到了’我做的事情，并且能够作为一家人共同分享非同寻常的时刻，我心里确实觉得很温馨。”- Ben Welland（加拿大）
-“Path 以其设计闻名，这家公司通过注重细节和仔细考虑所有元素，在这款应用程序中努力营造强烈的感觉。从按钮的阴影，到观看照片时某些功能的淡出方式，细节就是这样共同发挥作用，使得 Path 成为一款精美的应用程序。”- Eliza Kern (GigaOM)

在您所控制的精美应用程序中，与最了解的人和最在乎的人，随时分享您的生活。欢迎使用 Path！

特性：
- 记录您的私生活 - 可记录您的想法、睡眠和签到。
- 搜索生活瞬间 - 可通过关键字、季节、天气、日期、人员、地点等方式搜索所有记忆。
- 收集和分享使用媒体的瞬间 - 好友可以欣赏您正在欣赏的音乐，观看您正在观看的电影，阅读您正在阅读的书籍。
- 分享美好瞬间 - 通过某些为手机打造的最好用的滤镜和编辑工具，可捕获照片。
- 感受爱 - 亲友可以通过微笑、大笑、震惊、示爱、添加贴纸、发表评论等方式作出回应
- 进行多渠道发布 - 可通过向 Facebook、Twitter、Tumblr 和 Foursquare 发布 Path 瞬间到处发布生活瞬间。
- 传递消息 - 可以迅速、有趣和私密的方式，向亲友发送一对一消息或小规模群发消息。可通过言语、声音、位置、媒体、贴纸等发送消息！
- 新功能！进行私密分享 - 可决定与一个或多个好友私下分享 Path 瞬间
- 新功能！可通过选择一群好友创建Inner Circle，以便与之轻松分享生活瞬间。可过滤馈送的生活瞬间，仅查看Inner Circle中的生活瞬间。还可以更改设置，以便仅接收发自Inner Circle的通知。
- 新功能！Premium - 升级到 Path Premium 即可独享多种好处，包括摆脱对贴纸包的限制，摆脱对相机滤镜的限制，以及提前买到店中的新商品。此外，还可以加入让私生活始终私密且没有广告的潮流中。
- 贴纸 - 通过某些我们最喜欢的画师手绘的免费贴纸，点击一次即可表露更多心声。
- 商店 - 可购买更多贴纸和照片滤镜，以自定义 Path 并在消息中表露心声。
