On Board Diagnostic Code Reference contains well over 14,000 OBD I & OBD II trouble codes covering 50 different models of automobiles. Most models contain OBD I codes and all models support generic OBD II codes. This app does not connect to Bluetooth codes readers but is a great companion for code readers that do not contain internal databases. If the code you are looking for is not contained within our database then you can try searching the internet with the built in button. This app is compatible with all tablets and also features app to SD Card. This version is ad-free.

*定时器：用户指定的时间间隔和指定日的一周中，该规则应打开BT 。当规则被触发，蓝牙被激活两种模式：连续或取样。在连续模式下， BT是在整个规则所指定的时间间隔开启。抽样BT模式开启时均匀的时间间隔，直至规则到期或设备连接到一个配件。

* WIFI：用户指定一个WiFi网络， ，采摘从列表。当设备连接到指定网络AutoBluetooth开启或关闭BT取决于你所做的选择，同时配置规则（ “开”和“关”是唯一的选择） 。用户还可以指定日的一周中，规则是活跃。

* NFC （仅Pro用户） ：用户指定的时间间隔和指定日的一周中，规则应打开BT上，为的是定时器规则。在这种情况下，规则的检查，只有当一个NFC标签读取设备。

*地理围栏（仅Pro用户） ：用户指定，如果他想有BT或达到或离开某一区域时。该位置是在移动设备的当前位置，和该区域有一个100米的半径范围。进入或离开这些区域就会触发规则。用户还可以指定日的一周中，规则是活跃。

*电池充电（仅Pro用户） ：用户指定天一周BT被激活，当设备连接到电源（墙壁插座或USB电缆） 。当设备断开电源BT被禁用。


*电池充电（仅Pro用户） ：用户指定天一周BT被激活，当设备连接到电源（墙壁插座或USB电缆） 。当设备断开电源BT被禁用。

*电话呼叫（仅Pro用户） ：用户指定，其中一周天BT当设备收到一个电话必须acivated的。当通话结束BT被禁用

自动蓝牙是能够检测如果设备连接，可以等待其断开前关闭蓝牙。当设备连接在一个定时器活动，并保持连接后，在指定的时间，这是特别有用。
