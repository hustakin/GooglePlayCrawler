Ever wondered if there is a way to know who is calling while you are driving and the Phone is in your pocket? Or who send you SMS while you are in another room and received an SMS?
Caller Name Talker speaks out the Incoming caller name and SMS sender name so that you can identify who is calling without looking into your phone. It can even read the SMS content for you. And the best thing is, it is totally FREE.

Please download Ad Free version of this app Talk Caller Name PRO (https://play.google.com/store/apps/details?id=com.pa.caller.donate) to donate developer.


Caller Name Talker uses the built-in Android text-to-speech engine to voice the caller or SMS Sender ID. This program will not work if your phone doesn’t have text-to-speech data installed in your phone. No need to worry about that! Caller Name Talker will redirect you to Google play to install text-to-speech data into your phone if your phone doesn’t have it pre-installed.
There are a wide range of options to customize this application in the way you like. Some of them are:-
• Temporarily disable this feature without uninstalling application.
• Options to play caller name only if the caller name is in contacts list.
• Adjust Volume – Use ringer volume or use custom set volume.
• Turn off caller name alert feature if Phone is in silent mode.
• Configure time interval to repeat playing caller name.
• Flip phone to stop caller name alert while phone is ringing.
• Customize playing message by adding message before and after caller name. Eg:- John Doe Calling
• Alert SMS Sender name while receiving an SMS.
• Customize playing message when receiving SMS by adding message before and after SMS sender name. Eg:- SMS From John Doe
• Change voice speed and pitch settings.
• Option to test current voice speed and pitch by using custom message.
• Read SMS content.
• Option to switch between languages supported by Text-to-Speech Engine in your phone.
