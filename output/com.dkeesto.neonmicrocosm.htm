说明
This live wallpaper simulates a swim through a 3D colony of glowing microbes. Neon Microcosm is descended from the original Microcosm, and features a more intense color scheme and depth-of-field effects.

This full version lets you tap the screen to spawn new cells. It also includes several more color themes, and the option to select any colors you like to create your own custom theme.

If you have purchased the full version of Microcosm, then you don't need to purchase this! Microcosm unlocks all of the extra features in Neon Microcosm Free.

Neon Microcosm uses some special effects and is intended for high-end phones and tablets. If it is not running smoothly for you, please try reducing the number of particles in the settings, or turning off the depth-of-field effect.

Please try the free version of this live wallpaper before purchasing! If you have any problems at all, just send me an email at contact@cyphercove.com.
