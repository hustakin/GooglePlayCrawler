This is NOT androidlost. Do NOT install this unless you have a lost tablet without SMS connection and cannot start the registration process.

This app will wake up the registration process on the androidlost app when ever a phone call is made, an SMS received, battery is low, a package is added, removed or changed.

There is no graphical user interface for this app. So first install this app, then androidlost, wait a few minutes and finally log in to www.androidlost.com. If all goes well you should get a "Congratulations..." message.

Please note that there is no guarantee that the registration will take place. If the thief has blocked your account (if your phone is locked and he has tried three times) or has done a factory reset you will not be able to install any app.

After a successful registration you should uninstall this app. Otherwise this app will test if the app is registered correctly all the time.
