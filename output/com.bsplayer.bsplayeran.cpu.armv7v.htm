這是少了廣告、多了功能的 BSPlayer 完整版。

BSPlayer 是專為Android智慧型手機和平板設計的影音播放器，有著硬體加速影片解碼、自動搜尋字幕，以及自區域網路SMB共享且具緩衝的影音串流播放。

主打特色：
- 相容於Android 4.2 Jelly Bean MR1
- 沒有廣告
- 音訊前級擴大 (音量可提高至500%)
- 子母畫面播放 (音訊和影片)
- 影片硬體加速 --不僅顯著提升解碼速度還更加省電，支援雙核心與四核心CPU*
- 一手包辦幾乎所有影音格式，例如：avi, divx, flv, mkv, mov, mpg, mts, mp4, m4v, rmvb, wmv, 3gp, mp3, ac3等... 以及串流媒體，像是RTMP, RTSP, MMS (tcp, http), HTTP Live stream, HTTP
- 長寬比調整與縮放
- 多種音頻流和字幕的選擇
- 自訂個人化手勢，比方 快轉、跳轉、亮度和音量控制、以子畫面播放
- 支援播放清單和各種循環模式
- 可使用藍芽鍵盤
- 自訂聲音位移、播放速度、手勢和音高
- 適用外部與內嵌字幕 ssa/ass, srt, sub. txt...
- 自動尋找字幕 (行動上網或WiFi必須開啟)
- 直接經由WiFi播放您LAN(區域網路)中已共享設備或資料夾的影音檔 (諸如外部USB裝置、Samba(SMB/CIFS)共享設備、PC中共用的資料夾、NAS伺服器(Synology及其他品牌) --您根本不再需要辛辛苦苦轉檔並複製影片到手機或SD卡啦~~
- 直接播放RAR檔中的影音內容 (免解壓縮)
- 支援USB OTG，可參考： 以及更多相關APP

****** 授權疑難排解 ******
- 如果付費並安裝這個APP之後卻接到授權失敗通知，這是由於需要花一點時間在Google授權伺服器上記錄購買。此情形會在幾小時內解決，或請您嘗試重新安裝並重新開機。
- 如果下載被拒絕，表示您的信用卡授權已經失效。請更新您的信用卡資料(時常是因為住址不符)並再重新試一次。
- 若Play Store顯示"與您的裝置不相容"，請試著清空您Google Play和APP的快取並重新開機；然而也可能是系統版本不符造成。
- BSPlayer使用Google的標準授權服務，所以當第一次使用時，請保持WiFi或行動上網開啟；之後便不需要用到網路。再者，現有使用者遇到授權問題時，您可以試著清除APP資料並在連上網路後執行APP。如此一來，應該能夠授權您的APP。

假如您有任何關於授權的疑問，請寄email： sales@bsplayer.com 我們將為您解決。

這個解碼包支援ARMv7 with VFP and NEON，屬於其他CPU架構的請下載適合的解碼包。程式會告訴您該安裝哪一種解碼包。

小提醒：當回報錯誤時請附上您愛機的品牌和型號。也可以寄email給我們告知更多細節： android@bsplayer.com。我們正為著使用者竭力改進這個播放器，由衷感謝您的意見回饋。

* 硬體加速的支援視您手上裝置的影片解碼能力而定。在某些HTC的型號上直向硬解播放可能會出問題(例如HD和其他的款式--這是硬體的關係)。而在一些裝置上(Samsung Galaxy S2)可能不是所有影片格式都可以縮放/延展。

此軟體使用LGPLv2.1協議下FFmpeg的原始碼，來源可於BSPlayer網站下載。
