===>主题简介
霓虹主题是一款由GO桌面开发团队为GO桌面加强版量身定制的主题类应用。其昏暗的背景色调配合霓虹灯绚烂的图标风格让手机桌面铺满城市夜空的灿烂。主题包含了应用图标、壁纸、文件夹、功能表皮肤等全部桌面元素，为手机桌面提供了一整套美化方案！
最新版GO桌面系列主题，是手机桌面的最酷装备！

===>亮点：
- 超过50个了精美图标，涵盖了安卓系统手机的主要应用！
- 专属主题壁纸，让图标不再寂寞！
- 全新文件夹界面体验，管理手机应用不再单调！
- 独特的功能表皮肤设计，让手机系统内外兼修！

===>使用方法：
1. 请确定已经安装最新版本的GO桌面EX。（搜索“GO桌面EX”进行下载）
2. 请运行GO桌面，在GO桌面中的点击MENU键—>点击“主题”进入主题设置界面—>选择相应主题。
3. 通过谷歌电子市场的搜索功能搜索“GO桌面EX主题”,发现更多精彩主题！
4. 如果喜欢截图中的天气时钟小部件，请在电子市场上搜索“GO天气”下载。
5. 如果希望获得截图中进程管理、联系人、短信等实用小部件，请在电子市场上搜索“GO桌面窗口小部件”下载，或者在GO桌面中长按屏幕—>点击“GO桌面小部件”—>选择需要的部件，直接下载。
6. 本主题由GO桌面开发团队设计及开发，推荐下载！

开关 GO桌面小部件
GO Launcher Dev Team
1
免费 收听 阅读 仅限移动设备 观看 已安装
GO桌面开关小部件是基于GO桌面的工具类应用。它可以快速设置Wifi、GPS、屏幕亮度、锁屏、蓝牙等功能。相对于其它普通的开关小部件，GO桌面开关小部件提供一种全新而简单的拖拽方式来让你任意改变各个开关在小部件中的位置。

关于无法卸载的问题：
我们使用锁屏开关，需要获取锁屏管理权限，激活后将无法直接卸载GO开关小部件；如果您需要卸载，请在“设置-安全-设备管理器”中取消GO开关小部件”

支持大小：
-4x1
-1x1

支持操作：
- Wifi、铃声、亮度、蓝牙、数据、GPS、同步、飞行模式、屏幕旋转、震动、锁屏、铃声音量调整，媒体音量调整
- 编辑开关的位置

如何卸载GO开关小部件
因为GO开关小部件有锁屏的权限，如果用户想卸载它，请先执行下面的步骤：
在GO桌面主屏按“Menu”键 －> 选择“系统设置” -> 选择“位置和安全” -> 选择“选择设备管理器” -> 确保GO开关小部件没有被勾选。接下来你就可以按照正常程序卸载GO开关小部件了

如果出现无法WiFi开关不能正常关闭开关的问题，请执行以下步骤(不同机型的设置路径可能稍有区别)：

系统设置 -> 无线和网络 -> WLAN设置 -> 取消勾选”WLAN” -> 再次使用开关小部件的WiFi开关打开无线网络

如果出现无法蓝牙开关不能正常关闭开关的问题，请执行以下步骤(不同机型的设置路径可能稍有区别)：

系统设置 -> 无线和网络 ->蓝牙设置 -> 取消勾选”蓝牙” -> 再次使用开关小部件的蓝牙开关打开蓝牙功能

如果出现锁屏开关无法正常使用，请执行以下步骤：

系统设置 -> 本地与安全（或者安全）-> 选择设备管理器 -> 把“GO 开关小部件“取消打钩 -> 再次使用开关小部件的锁屏按钮进行锁屏


怎样添加GO桌面小部件？
-长按GO桌面屏幕上的空白位置，点击GO桌面小部件，再选择你想要添加的小部件。添加前请确保当前屏幕有足够的空间摆放小部件。
