说明
Depth of Field Calculator for Photographers. Calculates Hyperfocal Distance, Near Focus Limit, Far Focus limit, and Depth of Field(DOF).

Includes
1)Simple Mode
2)Advance Mode
3)Reverse Mode
4)Av Mode

Simple Mode, lets user select most common settings using drop down menus.

Advance Mode, lets user manually enter Focal Length, Aperture and Distance. Ideal if you are using uncommon focal lengths. Accessible through menu

Reverse Mode, takes Near Focus and Far Focus distance values and calculates all the possible combination of focal lengths and apertures to achieve the entered values. Accessible through menu

Av Mode lets user select Camera, Focal length, and Subject Distance, and calculates resultant depth of field against all aperture values ranging from F1 to F128.

Supports most SLRs, DSLRs and Compact Cameras from Canon, Nikon, FujiFilm, Kodak, Olympus, Panasonic, Pentax, Samsung, Sigma, Sony, Four Third System, 35mm Film, Medium Formats, and Large Formats.

User selectable measurement units include "Meters", "Centimeters", "Feet" and "Inches". Calculates aperture from F/1.0 to F/128, focal length from 4mm to 1200mm, and subject distance from 1cm to 5km.

Last used settings are automatically saved.

And:
