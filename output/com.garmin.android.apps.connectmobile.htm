说明

騎車從此不再孤單
透過即時追蹤功能，您的親朋好友可以即時追蹤您的比賽和訓練活動。 使用電子郵件或社交媒體邀請追蹤者，讓他們也能在 Garmin Connect 追蹤頁面上檢視您的即時資料。 追蹤者收到您的電子郵件邀請後，就可以在地圖*上追蹤您的位置，並即時追蹤活動已進行的時間、距離、速度和高度。 您若使用 ANT+ 配件 (例如速度/節奏感應器或心跳速率監視器)，追蹤者也可以檢視這些資料。

上傳活動，下載路線
Garmin Connect 行動版可無線上傳已完成的活動，讓您能立即儲存、分享和分析騎乘活動中的每個層面。 您可以選擇手動或自動上傳活動。 一旦將訓練張貼至 Garmin Connect，您的「連線」朋友就可以檢視這些訓練並發表意見。 您也可以使用您的社交媒體網路進行分享。


這個應用程式還可以讓您搜尋您的路線和訓練，並直接傳送到您的 Edge 510 或 810。 之後，您就可以使用這些路線和訓練來導航，或透過 Virtual Partner® 功能與自己先前的記錄比賽。

掌握天氣狀況
將 Edge 510 或 810 與您手機上的 Garmin Connect 行動版配對，以在您的 Edge 510 或 810 上直接取得即時天氣狀況、預報和警報。 將您做決策所需的資料準備齊全，確保您手機的安全並保持乾燥。

*請小心使用。 任何時候若透過 Facebook® 或 Twitter® 張貼文章，任何可以存取您的 Facebook 或 Twitter 個人檔案的人都能取得該連結。

You will need one of the following hardware configurations to connect ANT+ devices:

FAQ:
Why is this pre-installed on my phone?
This service is pre-installed as a system application on phones that provide built-in ANT wireless support to help enable the full feature set of your phone. This is comparable to other system applications such as WiFi, Bluetooth, Camera, Contacts, Phone, etc. except with the added benefit of it being updatable to the latest version without a firmware update. In the pre-installed scenario, Android will not let you uninstall the service as it is considered a "system app”. However, even if you don't plan on using ANT, you should know that the ANT Radio Service takes up very little space and will only run if it is invoked by an application.


Why does this need the ‘Network communication- Full Internet access’ permission?
The ANT Radio Service will never download or upload any data over the internet. This permission is only required because it is possible to install the ANT Radio Service on an Android Emulator (ie: for development purposes) in which case it requires access to internet sockets to communicate with an ANT USB Stick on the host Windows PC using the ANT Android Emulator Tools. This communication requires the INTERNET permission but only communicates with the local PC.


Why does this need the ‘System tools- Modify global system settings’ permission?
This allows initialization of whether ANT should be turned off or kept on when Airplane mode is entered.
