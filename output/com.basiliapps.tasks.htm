说明
"Tasks" is the ideal companion for your Android smartphone or tablet if you manage todo, shopping or other lists in Microsoft® Outlook. Intuitive to use with many powerful features like reminders, search and grouping it keeps your task list organised and in sync with your Microsoft® Exchange server.

★ What the app needs
Tasks works on Android devices from version 2.2 onwards with tablet support for Android 3.0 or later.
The app can synchronise with any standard Microsoft® Exchange Server installation (version 2007 or later) that has the "Exchange Web Services" (EWS) enabled. Office 365 works well, as do most hosted Microsoft Exchange Servers.
The app can not synchronise directly with Microsoft Outlook and is not compatible with Microsoft® Exchange Server 2003, Microsoft ActiveSync, Live.com, Zimbra or Kerio Connect.

★ Which tasks get synchronised
So far the app synchronises active tasks from the default tasks folder of one Exchange Server account.

★ Under active development and fully supported
Further enhancements are planed and we fully support the app. Whilst we appreciate reviews on Google Play, please contact us via the inbuilt "Contact Support" function or our email address below so that we can help you resolve the problem.

★ Android 4.1 users
Please also install our free "JB Workaround for Tasks" which addresses a defect in Android 4.1 (Jelly Bean) that can result in the loss of account settings after a reboot.

★ Permissions explained
We aim to use the minimum possible set of permissions. The account and system permissions are used to integrate the app with the Android account management and synchronisation system.

Please visit our website where you'll find a full tour of the app as well as an FAQ page.
