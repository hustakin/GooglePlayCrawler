Welcome to Navfree!

Navfree is a completely free navigation app that offers turn-by-turn directions, spoken and on-screen instructions, offline map use and address search and live search via Google and Microsoft Bing, among many other features.

The maps it uses are available for free download from the Navfree Store. The map data is stored on your device so there is no need for a constant data connection while driving. It also allows you to browse maps anytime, anywhere without paying for data connections, which means no costly roaming charges when abroad.

It uses high-quality, open-source map data from OpenStreetMap that has been created and is continually improved by as many as 400,000 users around the world. If you see a problem, you can help to improve the quality of the maps by signing up to www.openstreetmap.org.


STANDARD FEATURES OF NAVFREE
LIVE MAP DISPLAY
- Browse maps by dragging and scrolling.
- On-board maps - no data connection required for map browsing, address search or navigation.
- Multiple guidance displays available - toggle between 2D, 3D and safety screen mode.

USER-FRIENDLY INTERFACE
- Intuitive menus and large buttons designed for easy use with touchscreen devices.
- Day and night maps – switches automatically to low glare mode for safer night driving.
- Music volume automatically fades out when voice guidance is played.
- Automatic zooming at junctions.

EXCELLENT ROUTE GUIDANCE
- Fast routing engine allows you to calculate long routes in seconds.
- Automatic rerouting – if you miss a turn, Navfree reroutes you automatically.
- Demo route plan allows you to preview your route in real time before you leave.
- Direction lists allow you to check each step of your journey.
- Dedicated pedestrian navigation.
- Several routing options: fastest, easiest, most economical, etc.

DESTINATION SEARCH
- Search by the exact address or postcode of your destination.
- Search instantly by selecting any point on the map and pressing “Route here”.
- Simple POI search available (points of interest: cafés, restaurants, museums, etc.) without data connection.
- Use Google directly from Navfree. Search results appear instantly on the map interface, and then it takes only one click to route there.

Maps available: Canada, Mexico, Latin America.
