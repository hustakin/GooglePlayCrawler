说明
Data Monitor application helps the smart phones users be aware about the data traffic over Sony Ericsson Android phones and avoid big surprises and fear of high data bills. The main propose of this application is helps user keep track
of data traffic between his phone and the GPRS/3G network. Using Data Monitor, user can set alerts to his counters (sent data, received data and total data) and be notified when the amount of data set to that counter have been reached.
数据监控应用，帮助智能手机用户超过索尼爱立信Android手机的数据流量，并避免大的意外和恐惧的高数据法案。这个应用程序的主要目的是帮助用户跟踪
他的电话和GPRS/3G网络之间的数据流量。使用数据监控，用户可以设置提醒他的计数器（发送数据，接收到的数据，总数据），并设置该计数器的数据量已经达到时通知。
