How to install fonts
-------------------------------------
To install fonts, place the font files (***ttf or ***.otf) into the fonts folder, /[sdcard]/Phonto/fonts.
'Phonto' and 'fonts' folders will be created automatically after Phonto launched.

In addition to that, you can install fonts from file management app like ASTRO file manager.

1. Download font file from online. Google "FREE FONT".
2. Unzip if you downloaded zip file (***.zip).
3. Open font file (***ttf or ***.otf) in Phonto from file management app.

Dropbox http://db.tt/4RiIoRb7 allows you to easily share font files between PC and smartphone.
