1. 천지인 더블 키보드는 개인정보를 수집하지 않습니다. 경고문구는 구글에서 모든 키보드 앱 다운로드시 공통으로 적용되는 문구입니다.
2. 앱을 다운 받은후 반드시 두가지 - "환경설정" 및 "입력방식 변경" - 를 해야 사용할 수 있습니다.
(자세한 내용은 본 설명문 또는 네이버에서 "천지인 더블 휴대폰 설치 방법"을 검색하세요)
3. "천지인 더블 키보드" 애플리케이션에는 "천지인", "천지인 더블", "쿼티", "쿼티 더블" 4개 방식이 포함되어 있습니다. ( 키보드 설정은 한/영 자판 1초정도 "꾹" 누르면 키보드 설정에서 선택할 수 있습니다. )

[ "천지인 더블 키보드" 애플리케이션 설치 및 사용방법 ]
-> 설치 및 사용방법은 다른 한글 입력기와 동일합니다.
-> 중간에 나타나는 팝업 문구는 일반적인 사항이니 안심하셔도 됩니다.

첫째, "천지인 더블 키보드" 애플리케이션을 다운로드 합니다.
둘째,
핸드폰 환경설정(톱니모양 아이콘) -> 메뉴중 "언어및키보드
또는 언어및입력 터치 -> "천지인 더블 키보드"에 체크
셋째,
메모장,메시지,카카오톡 등 입력창 -> 텍스트 액정 화면을 1~2초간 "꾹" 누르기(진저 브레드 버전) 또는 휴대폰 상단의 키보드 모양의 아이콘을 아래로 드래그하기(아이스크림 샌드위치 버전) -> 입력방식 선택 -> "천지인 더블 키보드"로 변경 체크 하면 사용할 수 있습니다.
. 팬택 베가 기종의 경우 한글 키패드 좌측 하단에 있는 "1 별표" 키패드를 꾹 눌러서 바꿔야 할 수도 있다.


[ 천지인 더블 키보드 사용방법 : 모음의 "필기 획순"을 고려하여 직관적으로 설계되었습니다 ]

2. 쿼티 더블 키보드를 이용하면 훨씬 편리합니다.
가 모음 ( ㅐ ) ( ㅔ ) 가 없고 대신 그 위치에 모음 ( ㅘ ) ( ㅝ ) 가 있습니다.
나 모음 입력시 "왔다 갔다" 하지 않고 "제자리"에서 더블 터치만 하면 오타가 나지 않습니다.

(본 한글 키보드 어플리케이션은 삼성 단말 전용입니다.)

모아키는 터치스크린 기반 단말에서 한글 입력을 가장 쉽고 빠르게 입력 할 수 있도록 제공하는 키보드 프로그램입니다. 자음을 누른 후 간단한 제스쳐 만으로 한글의 자음과 모음을 한 번에 입력 할 수 있습니다. 보다 빠르고 정확하게 입력할 수 있는 모아키를 지금 경험해 보세요. (* 버전 1.0.111117부터는 긋기로 모든 모음을 커버 할 수 있게 됨에 따라, 기본모드가 긋기 모드로 변경되었습니다. 기존 팝업모드를 사용하고 싶은 경우 설정에서 활성화 하시거나 Space키를 길러 눌러 사용 하실 수 있습니다.)

"천지인 만큼 쉽지는 않습니다. 하지만 속도와 편안함이 달라집니다. 지금 분당 100자(300타)에 도전해 보세요!"

*설치된 모아키 타자프로그램을 이용하면, 현재 자신의 입력속도와 순위를 확인할 수 있습니다.

■ 모아키 설정 방법
처음 설치하시는 경우, 다운 및 설치 완료 후 "열기"를 누르시고 설정도우미 안내에 따라 설정하시면 됩니다.(또는 모아키 아이콘 선택)
(설정방법 : "환경설정" > "언어 및 키보드" > "모아키" 체크 > 하단의 "입력방법"을 누르신 후 "모아키" 선택)

* 공지 : 모아키 설치 과정에서 "모아키" 체크 시 띄워지는 사용자 정보수집 관련 경고 창은, 구글 안드로이드에서 어떠한 키보드 설치 시에나 일괄적으로 띄우는 경고 창으로, 실제 모아키와는 무관함을 알려드립니다. 모아키에서는 어떠한 개인 정보도 사용자 동의 없이 수집하지 않음을 알려드리오니 오해 없으시길 바랍니다.

[설치 오류 발생 시]
기존에 설치된 모아키로 인해 설치가 안 되는 경우, 기존 모아키를 "환경설정" > "응용프로그램" 에 들어가셔서 삭제 후 재 설치를 하시면 됩니다.
