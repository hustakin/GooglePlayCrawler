说明
私密空间是超级强大的隐私保护应用，巧妙而秘密地隐藏您不想让别人看见的联系人，短信，电话记录。

您是否曾担心别人偷看您的隐私短信？是不是曾在不适合的场合接到过尴尬的来电？是否担心某人有心或无心地查看您的通话记录？

“私密空间”帮您解决所有烦恼！

“私密空间”可以在您的移动设备上设置一个个人空间，您可以将私密联系人加入其中。在这之后，所有与私密联系人之间的短信(SMS和 MMS)和通话往来都会被保存在这个个人空间里面，而在系统的短信箱和通话记录里面不会看到。

另外，“私密空间”还可以隐藏自身的应用图标，给您无懈可击的隐私保护。

独特功能：
-- 别人不知道”私密空间“的存在，除了您自己
-- 保存私密联系人信息(短信和彩信)在私密空间里
-- 从系统中隐藏私密短信会话
-- 从系统中隐藏私密通话记录
-- 在不合适的时刻，阻挡私密来电
-- 伪装的短信提醒，您可以定义您的来信人伪装名，震动方式，提醒声音。打造只有您自己懂的来信提醒
-- 晃动手机快速关闭“私密空间”程序

===========
小熊來電通知是專為香港而設的來電通知及過濾程式。透過使用「廣告電話(香港)」的資料庫，可識別大量廣告電話 (包括漫游時的廣告來電)，可依需要提示或忽略該類電話。

此程式也可自行加入要過濾的電話清單，並直接經由 Wifi/3G 更新廣告電話資料庫。

廣告電話(香港)網站為 http://hkjunkcall.com

注意：
- 此程式現在含有掛斷電話功能，但不一定所有手機都能正常使用
- Android 2.3 系統因官方收緊 API 權限無法使用新方法進行來電靜音
- 一般使用此程式無需付費 (包括更新資料庫也無需付費)，只有在使用資料庫放在 SD 卡等額外功能時才需要，然而支持進階版本也是支持作者繼續開發最好的方法！

===========
以下系統由於屬於非正式的 Android 系統，此軟件並不保證可以使用，亦不會對以下系統提供任何支援：
- 小米 / MIUI
- 魅族 / Flyme OS

已知問題：
- 被生產廠商自訂過的系統可能無法使用新方法 (已知 LG 及 SE 有問題)
- 非官方 ROM 可導致更新時無法完成 (尤其是 Nexus One 的非官方 ROM)
- 3 3G及數碼通無法進行線上更新 (請改用 Wifi 更新)

** 此程式需要接收系統通知 (定時更新及來電)，所以是不可能安裝到 SD 卡的 (因為若在開啟 USB 磁碟模式時收到系統通知就會 Force Close)，別再來信要求支援 Move2SD。

注意！由於我不一定時常更新這個程式，你需要定期手動 (或排程) 在「偏好設定」裡進行資料庫更新！
