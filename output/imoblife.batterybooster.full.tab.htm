说明
電池優化衛士是一款專門為Android用戶設計的電池優化和電量管理工具。根據手機系統規格，電池優化衛士為您提供最精確的電池資訊，其中包括及時的電池容量，溫度，電壓，以及具有自動更新功能的電量跟蹤圖示等。電池優化功能包括智慧省電裝置以及管理WiFi、藍牙以及GPS的快捷設置，能夠輔助手機電池展現最佳狀態。

特徵：
1.及時的電池資訊（容量，溫度，電壓，充電狀態等）監測和電量使用跟蹤
2.支援桌面小部件（可通過功能表->添加桌面部件->電池優化衛士添加）
3.快速管理WiFi、藍牙、螢幕亮度以及螢幕待機時間
4.及時電池電量使用狀況監測
5.電池校正提示可幫您延長電池壽命
6.螢幕鎖定時自動關閉WiFi、藍牙
7.螢幕解鎖時自動開啟WiFi、藍牙
8.電量低時關閉WiFi、藍牙
9.電量低時語音提示設置
10.電量充滿時語音提示設置
11.電池溫度過高時語音提示設置
12.內置智慧省電裝置可更加有效的節省電量（根據記憶體加速器-Android RAM優化工具）

Homepage: http://www.downloadandroid.info

軟體功能：
- 即時記錄電池資訊
電池優化衛士即時記錄各種電池資訊，包括電池電量、溫度等並預測剩餘電量使用時間。

電池優化衛士提供了四種預設省電模式，不管你的設備使用習慣如何，四種模式都能有效的減緩耗電情況，延長電池壽命。除此以外，有特殊省電需求的使用者還可以添加個性化模式來優化省電效果。

電池優化衛士跟蹤並記錄每一款正在運行的程式的耗電情況，並以百分比形式展示出其耗電比例。通過這些資訊，你可以輕鬆找出設備中的耗電進程，並將其終止運行或者卸載。

通過電池優化衛士桌面Widget，你還可以輕鬆管理手機上的日常使用元素，包括WiFi（Wlan），藍牙，GPS，移動資料，飛行模式，螢幕超時，螢幕亮度，觸感回饋等等。

電池優化衛士還通過終止幕後程式來優化省電效果。對於不想被終止的幕後程式，你可通過設置-白名單來保護。安裝高級工作管理員軟體（記憶體加速器）的話，電池優化衛士會運行得更好，省電效果更佳！

Battery Calibration
NéMa
1
免费 收听 阅读 仅限移动设备 观看 已安装
Requirements: Android 2.1+ | Root permissions

Calibration needs to be done after flashing a new ROM, but you can calibrate any time you think your battery is miscalibrated. This program does it by removing the batterystats.bin system file. The OS generates a new clean batterystats file soon, thus any fake information from the previous ROM is removed.
It's suggested, but not necessary, to let the phone fully discharge after calibration, then charged to 100% without break.

The app shows your actual battery status, and only allows you to start the calibration (i.e. removes the file) when it's fully charged, unless you force earlier calibration.
The voltage meter can be useful too. Full charge is around 4200mV.
There is an option to beep when 100% is reached so you don't have to closley follow the changes.
