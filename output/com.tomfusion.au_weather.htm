Simple, concise, Australian weather ad supported app:
* BOM Observations and forecasts
* Customisable widget (2x1)
* 4x1 widget with 3 day forecast, pressure, wind and UV (where available)
* 4x2 widget with 3 day forecast
* 4x2 clock widget
* Widgets Jelly Bean lock screen enabled
* Multiple location widgets
* Temperature in status bar (black,white, boxed or colour via widget)
* Use nearest weather station or select specific from over 750 BOM weather stations
* UV index & warnings (where available)
* Radar loop (current station where available)
* Share weather (email, SMS etc.)
* Temperature/pressure graph for last 24 hours (where available)
* Last 24hr observations (where available)
* Low data usage
* Favourites list
* Pressure sensor support (Xoom etc.)
* Update frequency setting
* High contrast option (white text on main screens)
* Weather warnings (from details menu)
* Free, ad supported version (an ad-free, paid version also available)

IMPORTANT: This app uses the mobile network to determine location (not GPS) - if your location does not update see the FAQ to enable mobile network location or email support.

NOTES:
* This app only displays weather for Australian BOM weather stations!
* Custom ROMs are not supported (but mostly work) - adblocking will result in annoying message.
* The app is not optimised for QVGA or tablet screen sizes, however, it should work on most
* Support Android 2.1 was dropped
