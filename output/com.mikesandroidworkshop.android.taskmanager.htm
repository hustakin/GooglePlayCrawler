Frequently Asked Questions:
Q: How can I automatically dial a phone number entered in a task?
A: Type in a phone number in either the contact field or in the note field to display that phone number as a link when viewing it on either the 'Task Detail View' screen or the 'Alarm Notification detail' screen. Clicking on the link will automatically dial that number.

Q: How do I create a custom filter?
A: To save custom filters press the phones menu button while you are on the 'Sort and Filter Settings' window. A menu will pop up. Select 'Save Filter Settings As..'. Then type in the new filter name and press OK.

Q: Can ToDo Task Manager sync with Google?
A: To add a task to your calendar long press on a task in the task list until a context menu pops up. Then select the option 'Add to Google Calendar'.
The 'Toodledo.com Sync Add-on -Beta' add-on will sync with the web site Toodledo.com. On the Toodledo website there are third party tools to sync will Google ( http://www.toodledo.com/widget/google_gadget.php ). This should allow you to sync my application with Google indirectly.

Q: Can ToDo Task Manager sync with Outlook?
A: My application does not directly sync with Outlook. It provides the option to import and export tasks to Outlook using a CSV file.
The 'Toodledo.com Sync Add-on -Beta' add-on will sync with the web site Toodledo.com. On the Toodledo website there are third party tools to sync will Outlook ( http://www.toodledo.com/connect_outlook.php ). This should allow you to sync my application with Outlook indirectly.

Q: How can I search for a specific task in the to-do list?
A: The search feature is turned off by default. In the application settings select 'Searching, Sorting, and Filtering'. Then check the 'Quick Search On/Off' setting.
When quick search is turned on my application will filter the task list based on the text typed in the text field at the bottom of the task list screen.

Q: How do I create a new category?
A: To create a new category simple type in the new category in the category field on the task edit screen. The button next to the category field displays all of the existing categories in all of the other tasks.

Q: How can I enter additional information in a new task without having to click on the task in the task list?
A: Click on the add task button without entering a task title in the text entry field to automatically open the task creation screen. You can also set default values for most of the fields.

#描述
高级任务管理器简称ATMGR，这是一款用来杀任务或进程的应用。它可以列出所有正在运行的任务或进程，你可以选择需要杀死的任务，点击“停止选中任务”来杀死进程。它是一款任务管理工具，可以对运行中的人物进行管理操作，你可以长按任务某个任务或者应用，将会弹出一个弹出菜单提供更多的操作选项。

市场里有很多的任务管理应用，排名最高的是Advanced Task Killer简称ATK的应用，我们的任务管理器和他们有一些区别。我们的任务管理器比ATK列出的任务更全，因为我们采用的和系统设置里列任务一样的算法，所以我们的结果更加准确。另一个终哟啊的不同是我们的任务管理器还具有卸载应用的功能，可以帮助你卸载没用的软件。所有的任务管理工具一半都有一个小部件，可以点击杀任务，我们的小部件有一个独特的功能，它可以动态显示当前的可用内存，不过这个功能不是默认开启的，需要在设置里启用。

安卓2.2系统改进了进程管理机制，导致了所有的任务管理器无法杀死服务或者任务栏图标。如果你想要彻底杀死一个任务，你可以长按住这个任务，然后选择“强制停止/详细信息”，你将会看到一个该任务详细信息的界面，点击“强制停止”即可。你也可以通过按“Menu”，点击“服务”，进入服务管理的界面，在这里你可以停止后台服务。

#常见问题
问：为什么有些应用杀了之后又会反复出现？
答：很多应用会被某些系统事件启动起来，任务管理工具没有办法阻止他们重启，但是可以停止他们。所以我们建议你们在设置里启用自动清理功能，它将在每次锁定屏幕的时候自动杀死这些任务，这将会延长电池使用时间，释放内存。

Frequently Asked Questions:
Q: How can I automatically dial a phone number entered in a task?
A: Type in a phone number in either the contact field or in the note field to display that phone number as a link when viewing it on either the 'Task Detail View' screen or the 'Alarm Notification detail' screen. Clicking on the link will automatically dial that number.

Q: How do I create a custom filter?
A: To save custom filters press the phones menu button while you are on the 'Sort and Filter Settings' window. A menu will pop up. Select 'Save Filter Settings As..'. Then type in the new filter name and press OK.

Q: How can I search for a specific task in the to-do list?
A: The search feature is turned off by default. In the application settings select 'Searching, Sorting, and Filtering'. Then check the 'Quick Search On/Off' setting.
When quick search is turned on my application will filter the task list based on the text typed in the text field at the bottom of the task list screen.

Q: How do I create a new category?
A: To create a new category simple type in the new category in the category field on the task edit screen. The button next to the category field displays all of the existing categories in all of the other tasks.

Q: How can I enter additional information in a new task without having to click on the task in the task list?
A: Click on the add task button without entering a task title in the text entry field to automatically open the task creation screen. You can also set default values for most of the fields.

Q: why the permission to read contacts?
A: The phones contacts are read so this application can list the contacts in the contact field and create a button to link to the contact.

This application is an add-on that allows ToDo Task Manager -Pro to sync with the web site Toodledo.com.

WARNING: Please back up you tasks before using. This add-on will add, change and delete tasks.

REQUIREMENTS: This application requires that you have ToDo Task Manager -Pro installed and an account on Toodledo.com.


To use Toodledo.com Sync Add-on:

First go to the Toodledo.com web site and sign up for a free account.
Next open ToDo Task Manager -Pro and backup all of your tasks to your SD card, remember better safe than sorry.
Open this 'Toodledo.com Sync Add-on' application and click the Login button.
Enter the email address and password you used to sign up on Toodledo.com on the Login screen and click the OK button. You should see your user id displayed on the main screen with the words 'Login Successful' below it.
Now that you are logged in click the sync button to sync the tasks in ToDo Task Manager -Pro with the tasks on Toodledo.com.
