说明
Desktop VisualizeR(DVR) makes shortcuts or widgets with the image specified or the icon of another application for the Home screen. You can change the icon when you create a shortcut.

Don't know the exact size for widget images? Purchase DVRImageCrop or DVRImageSlicer. Some of widgets in screen shots are created by these add-ons. Please try.

Usage:From Home screen press menu->add->Widgets or Shortcuts->Desktop VisualizeR

* The crash log will be sent to us using our web server through the internet.

** Simple backup/restore, a new function of ver 1.4.0, tutorial **
1. Put DVR widgets on your homescreen.
2. Launch DVR and select menu > "Backup widgets" to make backup.
3. Select menu > "Refresh Widgets" > "Edit Mode" to enter edit mode.
4. Tap each widget and change widget image, action and so on.
5. Launch DVR again and select menu > "Backup widgets" to make backup.
6. Select menu > "Restore widgets" and choose the backup created at step 2 for the first state of widgets.
7. Select menu > "Restore widgets" and choose the backup created at step 5 for the second state of widgets.

桌面可视化工具（DVR）指定的图像或其他应用程序的图标在主屏幕的快捷方式或部件。当你创建一个快捷方式，您可以更改图标。

不知道小部件图像的精确大小？购买DVRImageCrop或DVRImageSlicer。所建立的一些部件在屏幕拍摄这些附加产品。请尝试。

用途：从主屏幕上按菜单 - >添加 - >小部件或快捷方式 - >桌面可视化

*崩溃日志将使用我们的Web服务器，通过互联网发送给我们。

**简单的备份/恢复，版本1.4.0的新功能，教程**
1。将DVR部件您的主屏幕上。
2。启动DVR和选择菜单>“备份窗口小部件”，使备份。
3。选择菜单>“刷新小工具”>“编辑模式”进入编辑模式。
4。点击每个插件和更换部件形象，动作和等等。
5。重新启动硬盘录像机，并选择菜单>“备份部件”进行备份。
6。选择“菜单>”还原“小部件”，并选择在第2步中创建的备份部件的第一状态。
7。选择“菜单>”还原“小部件”，并选择在第5步创建的备份部件的第二状态。
