说明
Ultimate iOS7 Theme is a brand new fully functional theme designed for all Android launchers, including full theme support for GO Launcher EX, Apex, Nova, Next Launcher 3D, Smart Launcher, TSF Shell, ADW/EX, Action Launcher Pro, Atom Launcher, Holo Launcher HD, SS Launcher, and more!

Theme your Android device with the latest Ultimate iOS7 style, shock your iPhone or iPad friends / enemies with a complete custom theme inspired by the forthcoming Apple mobile Operating System!

ULTIMATE iOS7 LAUNCHER THEMES FEATURES:

+ Ultimate iOS7 Icon Pack - Individually styled iOS inspired icon pack, featuring over 2,060+ meticulously designed, flat and perfectly minimal Ultimate iOS 7 style icons.

+ Bonus Holo-Apple Style Icons - Icon pack includes four additional dock bar icon packs-in-one, Holo-Apple (Light) pack for use without launcher dock bar or Holo-Apple (Dark) version, in superellipse or new circle styles to allow for

perfect customization!

+ Ultimate iOS7 Wallpaper Pack - Sixty custom designed wallpapers inspired by the latest iOS 7 dynamic wallpaper styles. This exclusive set includes Flagship, with Vivid, Rain, Girl, Nexus, Sun Beam, Neon Green Android, and Key Lime

Pie Concept variants, an exclusive Jelly Bean iOS7 wallpaper, and metallic alternates in blue or purple colors. Multiple sizes available to perfectly fit your device!

+ Cloud-based Wallpaper Picker - Set wallpaper directly from the cloud without taking up space on your device. Save wallpaper to your device to use with 3rd party apps (such as 3D Live Wallpaper for parallax effects!)

+ Easy Wallpaper Switching - Built-in wallpaper picker, simply press Menu > Wallpapers > Scroll to Ultimate iOS7 > Set wallpaper to switch.

+ Full HD Support - Support for tablets and larger devices, featuring highest resolution XX-HDPI icon pack and full HD wallpaper set. Increase icon size using launcher settings to maximum for full HD experience.

+ Bonus - Ultimate iOS7 Clock Widget - Theme includes a built-in analog clock widget, to add to homescreen, long-press or add > Widget > Ultimate iOS7 AClock.

+ Full Launcher Graphics Support - All supporting graphics for TSF Shell, Apex Launcher, GO Launcher EX, ADW.Launcher EX, Next Launcher 3D, and more! Includes more than 500+ additional high quality custom designed Ultimate iOS7

graphics!

+ Theme Your Whole Device - Matching 'Ultimate iOS7' messenger theme for GO SMS Pro also available here in the Google Play store: http://goo.gl/RaBlT

+ Also available in exclusive 'Electric Blue' and 'Hot Pink' versions, available here on Google Play: http://goo.gl/LfhVJ

DIRECTIONS & INSTRUCTIONS:
This is a theme (like a skin), you must either apply it using built-in app, or you can apply it using your preferred supporting homescreen / launcher app settings (see support website for full instructions).

This theme fully supports Apex Launcher, GO Launcher EX / HD, Nova Launcher, ADWLauncher EX / ADW.Launcher, Next Launcher 3D, Action Launcher Pro, TSF Shell*, Atom Launcher*, Holo Launcher HD*, Smart Launcher* GSLTHEME, SS Launcher, and more!

*You must manually apply theme on these launchers, full instructions on support website.

To change wallpaper:
Press Menu > Wallpaper > Scroll to Ultimate iOS7 Wallpapers > Set wallpapers

To change dock bar (Holo-Apple) icons:
First, choose whichever app you would like to use and drag to homescreen.
Long-press on that app's icon and select Edit > press the Icon > (Swipe left) and choose Ultimate iOS7 icon pack > Select a transparent icon to use. Then just drag it onto your dock bar.

To switch homescreen icons:
Long-press on that app's icon > Edit > Press the icon > Choose Ultimate iOS7 icon pack > Select an alternate icon style!
