DigiCal 是一款时尚易用的行事历应用程式，可将你的所有约会与Google Calendar同步。在横向和纵向模式下有六种强大的行事历检视(日、周、日程、月、月文字和列表)可选择以实现最佳的用户体验。伴随最新的Andr​​oid Holo Light主题，DigiCal 为执行着Android 2.2或更高版本的智慧手机和平板电脑带来了更干净和直观的用户体验。 DigiCal 还自带了5个漂亮的小工具(列表、网格、日、日列表及月)。
 
DigiCal 创新的功能如智慧操作栏可轻松编辑、移动、复制、粘贴和删除事件。使用我们独特的内置位置搜寻，为事件添加一个位置变得从未如此简单。你仅需输入地名(咖啡店、商店、公园)即可，DigiCal 会自动查找地址并储存。使用此功能，Google Now (Android 4.1+)可知道你需要去哪里。它会查看当前交通，计画路线并在你应该离开时通知你。
 
当在日、周和日程检视中选择一个事件时，会显示一个事件快显，提供了事件详细资讯。这样可轻松完成所有的事件，而无需开关每个事件的快显。本应用还支援两指缩放和划动导航等手势。例如，你可以在所有的行事历检视中透过左右划动来快速查看日期。
 
发现DigiCal 如何帮助你保持有条理的生活并让你完成更多的事情!
 
✓ 六种独特的行事历检视(日、周、日程、月、月文字和清单)，包括横向模式
✓ 五个漂亮且可完全自订的小工具(列表、网格、日、日列表、月)。
✓ 与Android Holo主题一致的时尚且直观的使用者介面
✓ 与任何执行Android 2.2或更高版本的装置相容

✓ 内置位置搜寻可快速为事件添加位置
✓ 支持Google Now (Android 4.1+)，在该去约会时会通知你，可计画路线、查看交通并计算路途时间
✓ 事件快显可轻松查看你的每个事件
✓ 支援两指缩放和划动导航等手势
✓ 提醒: 稍后提醒、地图和导航
✓ 为日期、日期列表、网格和月份的小部件 (DigiCal+ only)自定义背景颜色
