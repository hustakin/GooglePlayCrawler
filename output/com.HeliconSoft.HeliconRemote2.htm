说明
Helicon Remote is a utility for tethered shooting compatible with all recent Nikon and Canon DSLRs.

USB OTG adapter is required if your device doesn't have full size USB socket!

Key features:
- Wi-Fi support (Nikon WT and Canon WFT modules, Canon 6D, Nikon D7100 etc.)
- Automated focus bracketing
- Extra long exposures - up to 32 min (all Canon cameras, all Nikon cameras except for D90 / D300 / D700 / D3)
- Advanced exposure bracketing
- Geotagging (on devices with GPS receiver)
- Full screen Live View
- Focus areas highlighting
- Live View noise averaging
- Time lapse shooting
- Video recording
- Burst/Continuous shooting

- Hyper focal distance and DOF calculator
- Live histogram (gray/RGB)


Please see more information at http://heliconsoft.com/heliconremote.html about desktop version of Helicon Remote

Device requirements:
- Android 3.1+
- USB host (USB OTG adapter is required if your device doesn't have full size USB socket)
- Touch screen

Note: Motorola Droid phones need externally powered USB hub/dock station, HTC devices don't seem to have USB host at all.

Camera compatibility:
Canon: 70D, 700D, 6D, 5D Mk III, 1D X, 7D, 5D Mk II, 1D Mk IV, 1D Mk III, 1Ds Mk III, 60D, 50D, 40D, 650D/Rebel T4i/Kiss X6, 600D/Rebel T3i/Kiss X5, 1100D/Rebel T3, 1000D/Rebel XS, 550D/Rebel T2i/Kiss X4, 500D/Rebel T1i/Kiss X3, 450D/Rebel XSi/Kiss X2

Depth of Field Calculator for Photographers. Calculates Hyperfocal Distance, Near Focus Limit, Far Focus limit, and Depth of Field(DOF).

Includes
1)Simple Mode
2)Advance Mode
3)Reverse Mode
4)Av Mode

Simple Mode, lets user select most common settings using drop down menus.

Advance Mode, lets user manually enter Focal Length, Aperture and Distance. Ideal if you are using uncommon focal lengths. Accessible through menu

Reverse Mode, takes Near Focus and Far Focus distance values and calculates all the possible combination of focal lengths and apertures to achieve the entered values. Accessible through menu
