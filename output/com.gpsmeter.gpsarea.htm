说明
This program allows you to measure areas of any size, with a precision marked by your own GPS. The larger the area, the better the measure. The perfect tool for field and farm work. Combine this App with the free program "gpmReport" for send the measurements to printer in one single step.

The main features are:

- Coordinates Lat/Lon, UTM
- Precise Calculation of the area using projection UTM
- Precise reading of the position of the GPS using averages
- Units of measurement: Metric system or Imperial
- Export data in ESRI Shapefile
- Export data in DXF
- Export data Google Earth KML
- Display measurements under Google Maps
- Scales and Zoom
- Computes distance between cursor coordinates of map.
- Email directly the measurement file.
- Email directly the exported files.
- Import SHP files into map (polygons, arcs and points).
- Set the size of font in screen.
- Languages: English, Deutsch, French, Russian, Spanish, Portuguese, Polish, Czech , Hebrew and Turkish.

This program is compatible to file level with "GPSMeter PDA", so that you can also use "gpmReport" to see and print your measures in your PC. See http://www.gpsmeter.com/index.php?page=gpmreport

You can also upload the *.gpm file directly to the GPS Visualizer online utility.

More info about http://www.gpsmeter.com/index.php?page=android

相機測量應用：Diskometer是一個容易使用和便捷的相機為對象的長度，寬度，大小，比例，角度，面積，弧長和尺寸測量，可以作為一把尺子或捲尺工具。

    這個Android應用程序進行遠程非接觸式，在使用Android手機或Android平板的平面測量。它使用的手機攝像頭和任何可作為參考與已知大小的圓形物體。

    如最常見的DVD / CD，錢幣等的參考對象是包含在應用程序中。此外，您可以使用自己定制的圓形物體做攝像頭測量已知大小。

diskometer有以下模式：

- FreeHand的模式 - 允許你在任何方向來衡量的對象。

- 視角模式 - 允許你在任何方向的對象來衡量長度，面積和角度，比較彼此的大小。

- 區域模式 - 面積計算器，可以測量不同形狀的面積

應用表明，在不同單位的測量結果：
米，毫米，厘米，英尺，英寸基於用戶設置。

這也算相對單位的對象的尺寸與尊重的參考對象。

你有一個選項，以使相機視圖與相機的圖像，而不是實時影像或負載從一個照相館圖片快照。


特點：
- 測量弧的長度，角度，半徑
- 角度測量
- 區域措施
- 長度，大小，尺寸，比措施
- 任何圓形物體測量的基礎
- 測量與不同的單位，公制和英制單位：米，厘米，毫米，英寸，英尺。
- （！），它可能只使用一小部分的參考對象。循環引用對象約束須圓圈標記只有3分。它可以讓你在大的參考對象，不適合相機視圖，以提高測量精度的照片放大。

根據距離這台相機措施的應用程序，可以達到毫米決議的對象。
