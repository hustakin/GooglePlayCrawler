Insere ou altera o código da operadora de longa distância sempre que você fizer uma ligação. Agora com suporte a múltiplos chips! Veja abaixo se o seu aparelho funciona!
Suporte ao nono dígito para todos os DDDs de São Paulo e Rio de Janeiro (assim que entrarem em vigor)!
Atualize todos os seus contatos com o nono dígito! O único que funciona em Android 1.5!

Você pode manter seus contatos cadastrados em qualquer formato que o programa se adapta e sempre disca corretamente.

Caso já tenha cadastrado a operadora em todos os seus contatos ele pode alterá-la na hora da discagem. Ótimo para quem está mudando de operadora de celular.

Também é possível diferenciar operadoras em chamadas para celulares, fixos ou chamadas internacionais.

Permite também transformar todas as suas ligações em chamadas a cobrar, caso você tenha um celular pré-pago e esteja sem créditos.

Suporte a múltiplos chips que funciona com a maioria dos aparelhos de marcas conhecidas (Motorola, Samsung, LG, etc.). Se o seu aparelho é chinês ou de uma marca totalmente desconhecida o recurso de múltiplos chips provavelmente não vai funcionar pois a grande maioria desses aparelhos não obedece certas regras do sistema Android (infelizmente não há nada que eu possa fazer quanto a isso - é um problema dos próprios aparelhos).

Usuários de MIUI: É preciso ativar uma permissão de discagem para que o programa funcione corretamente! Isso pode variar dependendo do seu aparelho, mas nas versões mais recentes do MIUI isso pode ser feito no aplicativo de permissões.

Se tiver algum problema para usar entre em contato. Respondo 100% dos e-mails e resolvo 99% dos problemas quando tenho as informações adequadas!

Para os mais paranóicos veja abaixo o motivo de cada permissão solicitada pelo aplicativo:

- LIGAR DIRETAMENTE PARA OS NÚMEROS DE TELEFONE
Como o objetivo do programa é justamente alterar o número na hora da discagem ele pede essa permissão para aparelhos que não fazem a alteração corretamente. Para esses casos há um método alternativo de discagem na qual o programa cancela a ligação atual e faz outra com o número correto.

- LOCALIZAÇÃO APROXIMADA (COM BASE NA REDE)
Essa permissão é usada para identificar o DDD no qual você está. Isso permite que o programa transforme as ligações em locais quando necessário. O uso do serviço de localização roda em segundo plano e é usado o mínimo possível.

- LER SEUS CONTATOS
- MODIFICAR SEUS CONTATOS
Um dos recursos do Operadora DDD é justamente a alteração dos contatos para o formato que você preferir, incluindo o nono dígito para celulares onde necessário. A alteração de contato também é feita temporariamente ao discar (e o contato retornado ao normal quando termina a ligação) para que certos aparelhos consigam identificar corretamente o contato do número que está sendo discado.

- REDIRECIONAR AS CHAMADAS EFETUADAS
O principal objeto do programa é justamente alterar o número para que sempre seja discado corretamente.

- LER STATUS E IDENTIDADE DO TELEFONE
Para fazer o serviço corretamente o programa precisa saber quando você está fazendo uma ligação. Também é necessário poder ler as informações dos chips para detectar quando o chip foi mudado, saber qual a operadora do chip e para saber quantos chips estão instalados no aparelho.

- EXECUTAR NA INICIALIZAÇÃO
Para garantir que o programa sempre saberá em qual DDD você está ele precisar ser iniciado logo que o aparelho é ligado. Não há tempo para fazer isso no momento da ligação pois pode levar alguns segundos para determinar o local. A execução em segundo plano é extremamente leve e só é acionada quando você está se movimentando.

- LER REGISTRO DE CHAMADAS
- SALVAR NO REGISTO DE CHAMADAS
Para manter o seu registro de chamadas sempre com o número e contatos corretos após uma ligação.
