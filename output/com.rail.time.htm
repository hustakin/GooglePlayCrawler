说明
Access railway time table anywhere, anytime without being connected to internet. This application is like a printed railway time table in your pocket.
1. Train list from source to destination for a day or week.
2. Passing by trains through station for a day or week.
3. Train complete route.
4. Application and database gets stored on SD card so main memory required for this app is very less.

[NOTE: Database file has to be downloaded. Size of DB file is around 3 MB. Request you to download the DB while having good internet connection like WIFI or 3G. With GPRS connection also it works but may be very slow. You can also check for updates. You can download the database to stay uptodate.]

Additional features:
* Train online status
* PNR status
* The complete travel app for India
* Included near stations and checkbox for Bangalore, Delhi, Mumbai, Kolkata, Chennai, Hyderabad, Tirupati, Trivendrum, Kochin, Goa, Lucknow, Varanasi, Patna, Nagpur, Chandrapur, Bhopal, Mangalore.
* Save paper and environment by using this electronic time table.
* We have seperate applications for local trains of Mumbai, Chennai, Kolkata and Hyderabad. Please download and use those applications if you need time table of local trains for those cities.
