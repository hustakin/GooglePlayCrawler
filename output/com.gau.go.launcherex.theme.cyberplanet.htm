GO桌面开关小部件是基于GO桌面的工具类应用。它可以快速设置Wifi、GPS、屏幕亮度、锁屏、蓝牙等功能。相对于其它普通的开关小部件，GO桌面开关小部件提供一种全新而简单的拖拽方式来让你任意改变各个开关在小部件中的位置。

关于无法卸载的问题：
我们使用锁屏开关，需要获取锁屏管理权限，激活后将无法直接卸载GO开关小部件；如果您需要卸载，请在“设置-安全-设备管理器”中取消GO开关小部件”

支持大小：

-1x1

支持操作：

- 编辑开关的位置

如何卸载GO开关小部件
因为GO开关小部件有锁屏的权限，如果用户想卸载它，请先执行下面的步骤：
在GO桌面主屏按“Menu”键 －> 选择“系统设置” -> 选择“位置和安全” -> 选择“选择设备管理器” -> 确保GO开关小部件没有被勾选。接下来你就可以按照正常程序卸载GO开关小部件了

如果出现无法WiFi开关不能正常关闭开关的问题，请执行以下步骤(不同机型的设置路径可能稍有区别)：

系统设置 -> 无线和网络 -> WLAN设置 -> 取消勾选”WLAN” -> 再次使用开关小部件的WiFi开关打开无线网络

如果出现无法蓝牙开关不能正常关闭开关的问题，请执行以下步骤(不同机型的设置路径可能稍有区别)：

系统设置 -> 无线和网络 ->蓝牙设置 -> 取消勾选”蓝牙” -> 再次使用开关小部件的蓝牙开关打开蓝牙功能

如果出现锁屏开关无法正常使用，请执行以下步骤：

系统设置 -> 本地与安全（或者安全）-> 选择设备管理器 -> 把“GO 开关小部件“取消打钩 -> 再次使用开关小部件的锁屏按钮进行锁屏

GO资源管理器是GO桌面EX的一款多媒体资料管理插件。使用GO资源管理器，您可以在功能表中快速查看手机内的图片、音乐和视频，并进行排序、删除、隐藏、播放等操作。GO资源管理器将带给您更好的多媒体资料管理体验。

功能：
1. 通用功能：对图片、音频和视频进行排序、分享、隐藏，支持多选。
2. 图片管理：
— 自带GO图片查看器
— 支持图片旋转、查看图片信息、设置为壁纸和联系人头像
3. 音乐管理：
— 自带GO音乐播放器
— 播放音乐时，通知栏显示音乐播放控件，提供快捷切换音乐和播放/暂停功能
4. 视频管理：
— 支持基础通用功能
5. 桌面相册插件（下一版本添加）

小贴士：
1. GO资源管理器仅支持在GO桌面EX 3.36及以上版本中可用（请搜索“GO桌面EX”下载最新版GO桌面EX）
2. GO资源管理器在GO桌面EX中有图片、音乐和视频管理快捷方式，添加方式为：长按桌面屏幕 → 在添加标签中点击GO快捷方式 → 点击图片/音乐/视频快捷方式
3. GO资源管理器可在GO桌面EX的功能表中打开，打开方式：点击功能表左下角资源管理图标 → 选择管理图片/音乐/视频

GO多屏壁纸是一款壁纸增强插件，插件包内预置6张精美壁纸。使用GO多屏壁纸，您可以为桌面的每个屏幕单独设置壁纸。

特性：
1. 为每个桌面屏幕单独设置壁纸
2. 可从GO多屏壁纸内选择壁纸，也可从手机内存中选择图片作为壁纸
3. 滑动屏幕时以渐影的效果显示每个屏幕的壁纸
4. 手势切换壁纸（下一版本添加）

小贴士：
1. GO多屏壁纸在GO 桌面 EX中拥有独特的功能入口（请搜索“GO桌面EX”免费下载）
2. GO多屏壁纸支持在其他桌面应用中使用
3.使用多屏壁纸请务必切换桌面的壁纸模式为滚屏模式（GO桌面:菜单 - 壁纸 - 裁剪模式 - 默认模式 ）
4. 如何使用GO多屏壁纸？
— GO桌面EX：长按桌面屏幕 → 出现快捷添加界面 → 点击壁纸标签 → 点击GO多屏 壁纸图标
— 其他桌面：桌面屏幕 → menu → 壁纸 → 动态壁纸 → 选择GO多壁纸
5.欢迎大家向GO多屏壁纸提供反馈建议，告诉我们你们想要的功能；我们将持续改进。
