说明
全台唯一適地性天氣預報系統
-提供全台8仟萬個氣象預報點(預報空間尺度約每隔數十公尺即有一預報點)
-提供逐時天氣預報(預報時間尺度最小為小時)
-預報資料每小時更新一次(可每小時查詢一次最近更新之天氣預報)
最近版本更新：1.恢復郵遞區號查詢功能 2.恢復日出、日落時間顯示 3.錯誤更正
P.S.若有問題或建議,歡迎傳送電子郵件給我們!

使用說明：

1.使用地點查詢設定預報地點：
(1)使用定位系統預報現在位置
(2)使用語音或鍵盤輸入詳細住址或地點名稱
(3)可設定10個預報地點
(4)使用郵遞區號查詢預報地點
2.使用手勢滑動選單以顯示：
(1)小時預報：預報1小時後天氣及12小時天氣
(2)36小時預報：每3小時為間隔預報36小時天氣
(3)七天預報：以白天、晚上為間隔預報7天氣象
(4)警報：提供中央氣象局發布之警、特報
3.天氣提醒功能：
提醒最近一次查詢地點之氣象
按［Menu］鍵可更改設定
(1)警報/特報：提醒中央氣象局發布之警、特報
(2)下雨機會：下雨或大雨機會
(3)紫外線過量：紫外線過量或危險等級
(4)易中暑：舒適度指標為易中暑
(5)寒冷：舒適度指標為寒冷或非常寒冷
(6)隔日溫差過大：今日高溫(低溫)與前一日之高溫(低溫)相差超過3度C機會
4.天氣桌面小工具功能：
提供桌面小工具(widget)功能，須配合天氣提醒功能以更新天氣資訊，安裝時，將它們添加到您的螢幕(請先確認螢幕是否還有空間)。要插入一個 widget，長按螢幕上的空白區域，並選擇“小工具”，然後選“台灣氣象(大4x2)”或“台灣氣象(小4x1)”。

本公司(台灣氣象)經中央氣象局許可，依「氣象法」發布氣象預報，提供在地一週逐時天氣預報，天氣狀況、溫度、舒適度、雨量、紫外線、日出、日落、風向及風速。

天氣預報資料由台灣氣象公司提供。
