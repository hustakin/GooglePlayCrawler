Dr. Oz 极力推荐，可以帮助睡眠的神奇应用。了解一下为什么 White Noise 会让人睡得更香。多种环境声音帮助你白天精神放松 、睡眠香甜。

你夜晚睡不着吗？是不是想在坐飞机时补个觉？你的新生兒是不是总在半夜醒来？使用 White Noise 好处颇多：

• 屏蔽干扰，帮助您睡眠
• 减轻、舒缓压力
• 安抚哭闹的婴儿
• 提高注意力并更好地保护隐私
• 舒缓头痛和偏头痛
• 掩蔽耳鸣

就算你在睡觉，你的大脑也会不断地扫寻和倾听各种声音。如果周围太过安静，烦人的噪音如滴水的水龙头和警笛都会打断你的睡眠。White Noise 可以制造出多重频率的声音来屏蔽噪音干扰，让你不仅入睡容易，而且睡得更香。

声 音 目 录

空调，飞机旅行，亚马逊森林，海滩的浪声，风声，蓝噪声，水中摇曳的船，褐噪声，篝火，行驶的车辆，猫呼噜呼噜，时钟报时，城市街道，干衣机，蟋蟀鸣叫，嘈杂的房间，洗碗机， 瓢泼大雨，夜间的蛙声，爷爷的旧钟表，吹风机，心跳，大雨滂沱，细雨鸟鸣，大洋海浪，转动的电扇，粉噪声，雨点落在车顶，暴雨，喷水的花洒，流水的水龙头，流动的溪水，雷雨, 藏族诵经碗，开动的火车，吸尘器，紫噪声，滴水，喷水系统，白噪音

White Noise 曾被多个电视节目介绍过，如Dr. Oz Show，Fox and Friends First，Late Night with Jimmy Fallon；各大媒体（NBC News Today，Health Magazine，The Washington Post，New York Times，CNET, Esquire，PC Magazine） 亦进行过报道；同时还受到数以百万计用户的一致推崇。

应 用 特 性

• 40 多个完美循环声轨以及额外的免费音轨，尽在 White Noise Market - http://whitenoisemarket.com/
• 后台播放功能让您可以同时使用其他应用
• 将多种噪声混录即可创作新的音景。可以像 DJ 一样调整音量、控制平衡并掌握节奏
• 具备多种色彩以及亮度控制的全屏幕数字时钟让它成为床头柜上最实用的物件
• 先进的闹铃和计时系统可以控制声轨淡入淡出，让你醒后倍感精神
• 媒体播放器和声音控制都可通过手势来操作，让导航更加容易
• 在声音目录中为自己最爱的音轨和混录加星标，查找更加方便
• 支持所有基于 Android 的智能电话和平板电脑
• 高级设置可以阻止通知等干扰，调整缓冲区域大小改善播放质量，调整音量、平衡和节奏，自定闹铃小睡时间等等
• 没有广告

放松精神。睡得更香。感觉更棒。

ANDROID 权限解释

READ_PHONE_STATE：如果有电话打入，White Noise 会暂时变为静音。这个权限只为此功能所用。
WAKE_LOCK：允许闹铃被激活时唤醒设备。
DISABLE_KEYGUARD：允许闹铃被激活时解锁设备，闹铃结束时重新锁定设备。
INTERNET：用来提供网络信息、新闻。
WRITE_EXTERNAL_STORAGE：下载的音轨会被写入到外部存储设备。
READ_NETWORK_STATE：检查是否可以获取网络信息和新闻。

Permissions - Lightning Bug currently requires the following permissions on your Android device.

Your Location – course (network-based) location
Only used if you enable Lightning Bug’s local weather feature which is disabled by default. If you enable the local weather feature, only your latitude and longitude are sent to a trusted weather reporting service.

Network Communications – full Internet access
Required for downloading extra content including sound, image, and program files from 1908 Media servers. If you have Lightning Bug’s local weather feature enabled, Internet access is also required for contacting a 3rd party weather reporting service.

Storage – modify/delete SD card contents
Required for storing and managing extra content including sound, image, and program files.

Phone Calls – read phone state and identity
Required so that scene and alarm audio can be ducked (partially to fully muted) during phone calls.

System Tools – disable keylock, prevent phone from sleeping

System Tools – send sticky broadcast
Lightning Bug uses ‘Sticky Intents’ as a part of its alarm system. The app broadcasts a single sticky intent whenever the alarm clock goes off. This is a standard permission among most android applications providing alarm clock features.

System Tools – automatically start at boot
Lightning Bug very briefly runs at boot to synchronize it’s alarm system with the system clock. This process is required to ensure alarms go off at the correct time and is designed to have minimal impact on your device’s startup time. This is a standard permission among most android applications providing alarm clock features.
