说明
RepliGo PDF Reader enables PDF viewing, annotating, printing and form filling. Review documents on the go by adding highlights, cross outs, underlines, sticky notes, arrows, lines, ovals, rectangles, freehand drawings and text boxes. Comments are stored as standard annotations in the original PDF file, just like Adobe® Acrobat®.

RepliGo PDF Reader also makes a great eBook reader with features like reflowed text, color themes, text-to-speech and brightness settings.

Detailed Features:
* PDF viewing, annotating, printing and form filling of files from your email, memory card, cloud services and the Internet
* Direct access to cloud-based PDF files stored in Box, Dropbox, Google Drive and Microsoft SkyDrive
* Add highlights, cross outs, underlines, sticky notes, arrows, lines, ovals, rectangles, freehand drawings and text boxes
* Adjust colors, border thickness, font size and the note for annotations
* Read documents easily with reflowed text, text-to-speech, color themes and brightness settings
* Scroll quickly through pages by swiping, tapping on the edge of the screen, or by using the volume keys
* Navigate long documents using bookmarks, comments, document links, go to page, search and thumbnails
* Click on email, map, phone and web hyperlinks to launch system applications
* Zoom in and out quickly by pinching or double-tapping on columns, images or tables
* Select text and copy it to the clipboard, or look up definitions using ColorDict
* Rotate documents into one of four orientations
* Open password protected files
* Send PDF files via email, Bluetooth and cloud services
* Galaxy Note II Multi Window support
* Keyboard and trackball support

RepliGo PDF阅读器，PDF浏览，标注，打印，表格填写。审查文件，在旅途中增加亮点，跨出局，下划线，即时贴，箭头，线条，椭圆形，矩形，徒手画和文本框。评论被存储为标准的标注在原来的PDF文件，就像使用Adobe®Acrobat®。

RepliGo PDF阅读器也使一个伟大的回流文本，颜色主题，文本到语音和亮度设置等功能的电子书阅读器。

详细特点：
* PDF浏览，标注，打印和文件的表格填写您的电子邮件，存储卡，云服务和互联网
*直接访问基于云存储的PDF文件箱，升降梭箱，谷歌驱动器和微软的SkyDrive
*添加亮点，跨出局，下划线，即时贴，箭头，线条，椭圆形，矩形，徒手画和文本框
*调整颜色，边框粗细，字体大小和注释说明
*轻松地阅读文件，与回流文本，文本到语音转换，色彩主题和亮度设置
*快速滚动页面刷卡，拍打在屏幕的边缘，或者使用音量键
*浏览长文档中使用书签，注释，文档链接，进入页面，搜索和缩略图
*点击电子邮件，地图，电话和网络上的超链接启动系统的应用
