MailDroid is an ad-supported Webdav/POP3/IMAP Idle Push email client written from the ground up and and not based on the stock Android email client.

MailDroid has all the features you could want from a mobile email client and it continues to be built around user feedback. With the ability for IMAP users to sync draft mails, sent mail, deleted mail based on when they are on wifi or mobile as well as pop3 preload capability, MailDroid was built with mobile data in mind from the start.
