说明
HUD Dash verwandelt dein Smartphone oder Tablet in dein persönliches HUD für pCars!

Features:
- Layout Editor zur komplett freien Gestaltung von Anzeigen und Layout!
- Über 50 verschiedene Anzeigen für Daten aus dem Spiel.
- Von der Ganganzeige bis zur Streckentemperatur alles dabei.

Einstellungsmöglichkeiten:
- Einheitenwechsel möglich für Geschwindgkeit,Temperatur,Benzin und Druck.
- Schwellenwerte für alle wichtigen Anzeigen, um Warnungen direkt zu erkennen.

Vorraussetzungen:
- mindestens Android 2.3.3 (empfohlen ist 4.1)
- Das Handy/Tablet muss im gleichen Netzwerk sein wie der Rechner auf dem das Spiel läuft
(Bitte lade dir meine HUD Dash Connection Test APP zum Testen der Verbindung!)
- .NET Framework 4 muss auf dem PC installiert sein
- Die jeweilige Windows EXE für dein Spiel gibts auf www.barzmobilesolutions.de
- Tutorial unter http://www.barzmobilesolutions.de/de/race_hud_instructions_pCars.html


Bitte bei Gefallen Bewertung abgeben oder mir eine Mail schreiben um Verbesserungsvorschläge einzureichen.
