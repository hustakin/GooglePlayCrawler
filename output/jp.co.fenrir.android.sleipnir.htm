说明
The one step browser that once you try you can't let go of, made in Japan!

【3 advantages】

★ Gestures adjusted by the millimeter for speed and precision

★ Tabs with lightning fast "Open" "Close" "Switch"

★ Customization to cut your daily browsing into a third

【Characteristic functions】

• Gestures
Tab switching, refreshing, displaying bookmarks, full-screen, all in 1 step.
　
• Hold And Go
Open pages one after another by holding down on links.

• Custom button / Custom panel
In the freely customizable User Interface, you can arrange your desired functions, settings, bookmarks(bookmarklets too) where you can find it in just 1 step.

• Tab flicking
Flick tabs downwards to close, up to manage groups, and up and then right to open.

• Scroll grip
Quickly scroll down to the area of the page you want to view by touching the scroll bar.

• Offline viewing
Open tab content is automatically stored so that even after you close the app you can instantly restore data. (※ Android 3.0 and later)

• View mode
When opening specified pages and sites, you can have them automatically switch user agent and open in another app. (※From the menu button)

• Extensions
Layout of pages can be changed, and special functions can be added using extensions. Download from the Extensions Gallery ( http://extensions.fenrir-inc.com/ ). Installing extensions can be completed by simply tapping the install button.

• Fenrir Pass
This is a free cloud service that can be used to automatically sync bookmarks with Sleipnir on Windows, Mac, iPhone and Windows Phone, and also use a number of Web services.
Supported services: Twitter / Facebook / Google+ / LinkedIn / Tumblr / Flickr / Readability / Instapaper / Pocket / Dropbox / Evernote / SkyDrive / Google Reader

【Sleipnir Mobile Black Edition について】

Sleipnir Mobile（無料）の機能に「検索エンジン選択」の搭載と、黒を基調としたアイコンと画面デザインを採用した有料アプリケーションです。

デザインの変更箇所は以下の3箇所です。
• アプリケーションアイコン
• アドレスバー
• 新規タブ追加時に表示される「いつものサイト」

※ 通常版から乗り換えの際は、バックアップ機能でタブ、ブックマーク、設定を移行することができます。
※ タブバーの色はタブを上にフリックすることで変更できます
※ 検索サジェスト機能は、検索エンジンが Yahoo! JAPAN に設定されているときにのみご利用いただけます。

これらの機能が必要ない場合は、無料の Sleipnir Mobile をご利用ください。
https://play.google.com/store/apps/details?id=jp.co.fenrir.android.sleipnir


【3つの魅力】

★ 速さと精度のために1ミリ単位で調整されたジェスチャ

★ 「開く」「閉じる」「切り替える」の素早さを極めたタブ

★ いつもの操作が1/3になるカスタマイズ性

【特徴的な機能】

• ジェスチャ
タブ切り替え、更新、ブックマーク表示、全画面化など多くの操作が、たったの1ステップで完了します。
　
• Hold And Go
あとで読みたいリンクは長押しするだけで、隣にタブを開いて読み込ませておくことができます。

• カスタムボタン／カスタムパネル
柔軟にカスタマイズできるUIで、お好きな機能・設定・ブックマーク（ブックマークレット）を1ステップで届くところに配置できます。

• オフライン表示
開いているタブの内容は自動的に保存されるので、一旦終了しても保存データから一瞬のうちに復元します。だから電波を気にせずいつでもどこでもすぐに確認できます。（※ Android 3.0 以上）

• タブのメモリを自動解放
たくさんタブを開いても自動的に古いタブのメモリを解放するので、100タブあっても軽くて安定しています。解放されたタブは次回表示するときにすぐ復元します。

• スクロールグリップ
どんなに長いページでも、スクロールバーをタッチして目的の位置まで一気にスクロールできます。

• タブ・フリック
タブを下にフリックするだけでそのタブを閉じることができます。また上にフリックでグループ管理、上から右にフリックで新規タブを開くこともできます。

• 人気トピックス
ニュースや人気記事、人気サイトの最新記事をまとめてチェックすることができます。

• 表示モード
指定したページやサイトを開いたとき、自動的にユーザーエージェント(User Agent)を切り替えたり別のアプリで開くことができます。（※アドレスバー長押しメニューより）

• ピックアップ検索
表示中のページや別アプリから受け取ったテキスト内のキーワードをタップするだけで検索することができます。（※メニューボタンもしくはアドレスバー長押しメニューより）

• エクステンション
Web ページの見た目を変更したり、特定の機能を追加することができます。エクステンションのダウンロードは、Extensions Gallery ( http://extensions.fenrir-inc.com/ )から。エクステンションのインストールは、インストールボタンをタップするだけで完了します。

• Fenrir Pass（フェンリル・パス）
Windows や Mac、iPhone、Windows Phone 上の Sleipnir と自動でブックマーク同期や、統合された様々なウェブサービスを利用できる無料のクラウドサービスです。
対応サービス: Twitter / Facebook / Google+ / LinkedIn / Tumblr / Flickr / Readability / Instapaper / Pocket / Dropbox / Evernote / SkyDrive / Google Reader

【その他の機能】

• 設定不要の広告ブロック（※ Android 3.0 以上）
• スマートフォンとタブレットそれぞれに最適化したデザイン
• お好みに合わせて柔軟にカスタマイズできるUI

• 他のブラウザで開く（※アドレスバー長押しメニューより）
