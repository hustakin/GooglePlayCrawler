说明
SwipePad是一個啟動器叠加面板。 你可以在任何應用中通過一次滑動啟動任何其他程序。

它使用起來非常簡單與直觀:
1. 碰觸預設的屏幕邊緣區域;
2. 移動你的手指到屏幕中央啟動控制板;
3. 在特定的區域釋放手指以啟動程序;

就是这样！ 這將給你帶來新的多任務體驗:)

注意事項:
★ 控制板可以放置任何應用，快捷方式，書籤，聯繫人等等。 通過 widgetpad插件，它甚至可以從任何位置啟動一個小工具！
★ 這不是一個桌面應用，桌面設置偏好不會被改變(與所有第三方桌面應用兼容，例如LauncherPro, ADW Launcher等)
★ 在SwipePad下的應用將保持運行: 不會中斷
★ 對內存與電量友好

关于无法卸载的问题：
我们使用锁屏开关，需要获取锁屏管理权限，激活后将无法直接卸载Next开关小部件；如果您需要卸载，请在“设置-安全-设备管理器”中取消”Next开关小部件”

如何卸载Next开关小部件

在Next桌面主屏按“Menu”键 －> 选择“系统设置” -> 选择“位置和安全” -> 选择“选择设备管理器” -> 确保Next开关小部件没有被勾选。接下来你就可以按照正常程序卸载Next开关小部件了

有些开关可能会在Next开关小部件升级后不能正常关闭或开启，请执行以下的步骤。
如果出现无法WiFi开关不能正常关闭开关的问题，请执行以下步骤(不同机型的设置路径可能稍有区别)：
系统设置 -> 无线和网络 -> WLAN设置 -> 取消勾选”WLAN” -> 再次使用开关小部件的WiFi开关打开无线网络

如果出现蓝牙开关不能正常关闭开关的问题，请执行以下步骤(不同机型的设置路径可能稍有区别)：
系统设置 -> 无线和网络 ->蓝牙设置 -> 取消勾选”蓝牙” -> 再次使用开关小部件的蓝牙开关打开蓝牙功能

如果出现锁屏开关无法正常使用，请执行以下步骤：
系统设置 -> 本地与安全（或者安全）-> 选择设备管理器 -> 把“Next开关小部件“取消打钩 -> 再次使用开关小部件的锁屏按钮进行锁屏

G+ page https://plus.google.com/113192992285762862111

Unlike other current WP7 launchers (Windows Phone Android, Metro UI), this one allows you to properly modify your start screen. Just long press on a tile and drag tile where you want!

This version is free, but does have an ad in application list. Ad free donate version is available if you don't like the ad.

Also supported:
- 2 x 1 tiles
- Original Windows Phone 7 backgrounds!
- Freely configurable tile colors
- Launcher options are available from Application List (press menu in app list)
- Layout animations
- Animated contacts tile! (still work in progress)

TODO-list:
- configurable animation parameters (soon)
- folders (in donate version now)
- clock tile of sorts
- more animated tiles
- facebook integration

Consider this as alpha quality at best. If it crashes, please email me or report the crash and we'll figure out a fix.

Recommended HDPI device, tested on Galaxy i9000.

MDPI devices should be now able to show things in correct scale, but performance and memory usage may very well be a problem in these devices.
