這個應用程序搜索與谷歌API的圖像。
龍點擊下載圖像或去網站，其中包含的圖像。

*搜索
只需輸入查詢，然後按按鈕來執行您的搜索。
有幾個選項可以在設置菜單的“菜單” - >“設置”按鈕調用。

可選項：
  - 文件類型：任何/ JPG / PNG / GIF / BMP
  - 站內搜索：搜索限制在指定的域。（前）yahoo.com
  - 彩色化：任何/顏色/灰度
  - 圖片大小：巨大的圖標〜
 /面/照片/剪貼畫/線條稿/ - 圖像類型：
  - 安全等級：主動/中/關

*預覽
點選您的搜索結果調用一個放大的縮略圖。
該縮略圖是模糊的，但你可以看到的圖像看起來像什麼，然後再下載。

*保存圖片
您可以通過點擊“下載”按鈕，長按您的搜索結果中調用下載圖像。
下載的圖像將被保存在一個文件夾命名為“GIMGS”在SD卡。

*轉到網站
您也可以選擇“走出去”，而不是“下載”網站，訪問的網站，包括你所選擇的圖像。

FAQ 常见问题解答
===
问：如何取消隐藏的图片和视频？
答：1.打开KeepSafe，选择图片和视频
2.在最下面的工具栏中，点击“取消隐藏”
3.确认取消隐藏操作

问：KeepSafe是如何工作的？
答：KeepSafe在您的手机上为您提供一个上锁的图片库。

问：这款应用程序隐藏视频吗？
答：是的，我们支持单个视频和多个视频的隐藏。而且，对于隐藏的视频数量，也没有限制。

问：取消隐藏的图片位置在哪？
答：取消隐藏后的图片位置还是在您隐藏图片的位置。

问：忘记密码，如何找回我的密码？
答：打开KeepSafe，然后长按住KeepSafe图标，直至出现密码为止...很简单！

问：我隐藏的图片能够在线存储吗？
答：不能在线存储。保险箱只能存储您Android手机中锁定在图片库的图片。

问：KeepSafe支持wmv格式的视频播放吗？
答：不支持，保险箱不能播放wmv文件。

问：KeepSafe最多可以隐藏多少张图片？
答：您想隐藏多少就隐藏多少。

问：它和图片锁相似吗？
答：不相似。KeepSafe是一个特殊的图片库，能够锁住您的图片，不过，您的图片就放在里面。

问：KeepSafe可以锁住视频吗？
答：可以。如果您把视频放到KeepSafe里，它们就会被锁到一个PIN键盘后面。

问：我可以把图片从此安全箱中移出吗？
答：可以。您只需点几下鼠标，就可以取消隐藏图片。

How to install fonts
-------------------------------------
To install fonts, place the font files (***ttf or ***.otf) into the fonts folder, /[sdcard]/Phonto/fonts.
'Phonto' and 'fonts' folders will be created automatically after Phonto launched.

In addition to that, you can install fonts from file management app like ASTRO file manager.

1. Download font file from online. Google "FREE FONT".
2. Unzip if you downloaded zip file (***.zip).
3. Open font file (***ttf or ***.otf) in Phonto from file management app.
