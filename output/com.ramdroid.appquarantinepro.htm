*Update for Android 4.3 users:*
User Manager only works with Android 4.2, and is now obsolete with 4.3
Google have integrated the new user profiles that basically do what User Manager has been doing.
Besides that, due to Google's changes the methods used by User Manager are no longer working!

Allows you to share apps between multiple user accounts. When the device is used by family members then you can easily disable apps they should not be using.

Furthermore you can monitor the running processes on each user account, and stop them. When you switch between users then the apps of all user accounts are still running in the background. After a while your device is not so responsive anymore. Now with User Manager you can fix this issue easily.

This is the free version that is limited to manage one guest user only.

Requirements and limits:

- Only works on Android 4.2 devices with enabled multi user support. The app will NOT enable multi user support if it's not enabled by default. Currently Google only supports multiple user accounts on tablets.

- Root access is needed! If your device is not rooted then you can't use this app.

- After you have done some changes the app needs to reboot your device. Sorry for that but at the moment there is no other option! I hope in the future I can offer a better solution without the need to reboot.

- Currently the app can only be used by the device owner.

- You can enable both free and paid apps. However apps with a license check are only working if the Google account they were purchased with has been added for this user account.
