==== 시간표(Simple TimeTable) 소개 ====
기존의 시간표들이 기능을 강조한 나머지 복잡하고 사용하기 어려워 간단하게 작성할 수는 없을까 하는 맘에 만들게 되었습니다.
시간표 정보를 보여주는 화면에서 해당항목을 선택하여 쉽게 글을 쓰듯 작성만 하시면 되는 아주 익숙한 시간표 입니다.

이런 취지를 이해해 주시고, 프로그램의 부족함과 개선사항은 메일로 보내주시기 바랍니다. 최대한 개선하도록 노력하겠습니다.

==== 시간표(Simple TimeTable)의 장점 ====
- 많은 사이즈의 화면을 지원 ( 작은화면부터 갤럭시탭 10.1까지 지원)
- 간편하고 편리한 사용법 (시간표만을 위한 심플한 기능)
- 빠른입력 (복사,편집 기능등을 제공하여 빠른 입력이 가능합니다.)
- 여러개의 시간표를 지원( 교사시간표, 근무교대표, 친구 시간표 등을 관리 가능합니다.)
- 다양한 색상지원 (사용자가 과목별 색상을 만들어 지정하실 수 있습니다.)

==== 기능 설명 ====
1. 시간표 기능에만 충실하며 한화면에 시간표정보가 나오도록 처리.
- 한화면에서 모든 정보확인이 가능하도록 구성함.
- 메뉴버튼을 통해 View,Edit,Copy, Delete,List mode기능을 제공하여 해당항목 수정,삭제가 가능합니다.
ㄱ. View - 보기모드
: 보기만 가능합니다.
시간표 편집 복사등으로 셀을 보호 할수 있도록 보기 모드로 관리 되고 있습니다.
ㄴ. Edit - 편집모드
: Edit 모드로 해야 각 셀의 편집이 가능합니다.
ㄷ. Copy - 복사모드 (toggle)
: Copy모드 설정후 복사할 셀을 선택하면 별표가 나타납니다.
이후 복사될 곳의 셀 을 선택하시면 복사가 됩니다.
다시 복사할 셀을 선택(별표)를 선택하시면 다시 다른셀을 복사 가능합니다.
ㄹ. Delete - 삭제모드
: 셀을 삭제할 수 있는 모드 입니다. 삭제할 셀을 선택하시면 삭제할까요? 팝업이 뜹니다. 예를 선택하시면 삭제 됩니다.
ㅁ. list - list관리
: 추가버튼을 통해 여러개의 리스트를 추가 할 수 있습니다.
삭제메뉴를 통해 불필요한 리스트 삭제가 가능합니다.
기본시간표 설정을 원하는 항목을 길게 누르시면 기본시간표 설정을 할 수 있습니다.

2. 여러 시간을 병합가능합니다.
- 한과목이 2시간이상 수업진행될때 작성 가능하도록 처리 하였습니다. 셀 병합으로 인해 한눈에 확인 가능합니다.

3. 시간표시 선택 지원
- 0교시 모드 지원,
- 교시표시, 시간표시, 교시+시간표시 선택 가능

4. 여러가지 색상을 지원합니다.
- 다양한 색상지원으로 과목별 색상처리가 가능합니다.

5. 여러개의 시간표관리가 가능합니다.
- 여러개의 시간표 작성이 가능하여 여자친구 시간표, 학원시간표, 시험시간표, 지난시간표.. 등을 관리 하실 수 있습니다.

6. 더블클릭으로 화면 확대가 가능합니다.

7. 가로보기 기능이 제공됩니다.

8. 위젯 설정.
- 풀화면 위젯을 지원하여 시간표목록 진입하지 않고 시간표 확인 가능.
- 시간표 작성이후에 화면 저장을 위해 캡쳐(왼쪽상단의 작은 동그라미)버튼을 누르시면 위젯갱신이 됩니다.
