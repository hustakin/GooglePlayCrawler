WebBrowser for SmartWatch is a mini webbrowser.
Only working with Sony SmartWatch and Sony SmartWatch 2.
If you have problems, set "problem" checkbox in settings or send me an E-Mail so I can help and fix them.

In this version you can do:

* Click on the links.
* Enter text in textbox (like search textbox).
* Zoom.
* Search with your search engine. (default: Google).
* Search with Wiki.
* Set home page.
* Change the user agent string.
* Browser Width / Height.
* On / Off display of images.
* LocalStorage.
* MobileBrowser Mode.
* Favorite menu (you can put your active page as favorite by longtouch on your chosen number in favorite menu or in settings) and swipe up to get more favorite.
* NEW: Custom Keyboard; user can change the layout in settings. (default: for custom keyboard; Cyrillic keyboard)
 * NEW: Added Basic authentication in settings.

* Scroll with Swipe Up/Down/Left/Right.
* Longtouch for menu.
* Quickbutton for Zoom Up/Down.
* Quickbutton for navigate back/forward.

In keyboard view:

* NEW: Swipe Up/Down to change keyboard (Org, Num and Custom Keyboard).
* Swipe left to remove.
* Swipe right to insert space.
* To cancel the input mode swipe left until all signs are gone then swipe again.

and more coming soon...


This App is designed for the Sony SmartWatch 1 and 2.
