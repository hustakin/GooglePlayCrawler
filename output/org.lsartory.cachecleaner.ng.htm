常見問題：
1.什麼是快取？
答：快取是一種將數據存儲在高速內存中，加速應用程式讀取速度，提升應用程式性能的機制。

2.為什麼要清理快取？
在android手機中，快取文件存儲在手機內部記憶體中（ROM），手機內部記憶體不同於SD卡，可以理解為電腦中的內存和硬盤，一個速度快一個速度慢。
因為手機內部記憶體的容量是有限的，如果快取過多，而沒有及時清理，則會導致內部記憶體不足，快取機制失效而降低應用程式的性能，並且不能安裝新的應用程式。

3.清理後會不會丟失應用程式的數據？
答：快取跟應用程式的數據不同，快取是臨時存儲的文件，使用後就失效了；而數據是應用程式需要使用的記錄，如游戲記錄，用戶信息等。
清理快取不會清理應用程式的數據，請不用擔心
清理快取也不會清除應用程式的歷史數據，如搜索歷史等汪，來電記錄等。如果需要清除歷史數據，可以使用我們提供的另一款應用程式“歷史記錄清理++”

It provides replacements for most of the utilities you usually find in GNU fileutils, shellutils, etc. The utilities in BusyBox generally have fewer options than their full-featured GNU cousins; however, the options that are included provide the expected functionality and behave very much like their GNU counterparts. BusyBox provides a fairly complete environment for any small or embedded system.

BusyBox has been written with size-optimization and limited resources in mind. It is also extremely modular so you can easily include or exclude commands (or features) at compile time. This makes it easy to customize your embedded systems. To create a working system, just add some device nodes in /dev, a few configuration files in /etc, and a Linux kernel.
