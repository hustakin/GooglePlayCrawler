说明
Xtm (eXtreme Task Manager) shares features of both Task Manager and Program Launcher.
You can switch all running programs smoothly, and launch applications easily.

Features

* Star button
No need to long press 'Home Key' to switch between apps.
Just click the button on the screen.
- Starred : switch (or launch) between your most favorite applications.
- Previous : switch (or launch) between two recent tasks.

* X button
Using X button (top right corner), you can use some action as follows:
- click : close current task (app)
- long click : close all task (app)
- drag toward left : restart current app
- drag toward left down : show program launcher
- drag toward down : show context menu

* App Launcher
Using X button drag toward bottom-left corner
- Favorite : You can launch your favorite applications, shortcuts
(navigation, music player, direct sms, direct call, internet bookmark, system setup shortcut, contacts, etc...)
- Recent : You can launch application that recently launched
- All : Show all installed applications

* Context menu
Using X button drag toward down, some useful function about current running application.
- Information : detail information
- Close : close current task
- Close All : close all task
- Restart : restart current application, move to the beginning.
- Uninstall : Show uninstall dialog
- Market : Go to 'Play store'

的XTM（极限任务管理器）的任务管理器和程序启动的功能。
您可以切换所有正在运行的程序顺利，轻松地启动应用程序。

特点

*星号按钮
  无需长按“Home键”的应用程序之间进行切换。

   - 上一篇：最近的两个任务之间的开关（或启动）。
  
* X按钮
 使用X按钮（右上角），你可以使用一些行动如下：
   - 点击：关闭当前的任务（应用程序）

   - 向左拖动：重新启动当前应用程序
   - 拖动对遗留下来的节目发射器
   - 将走向：显示上下文菜单

*应用程序启动
  使用X按钮拖向左下角
   - 收藏：您可以启动您最喜爱的应用程序，快捷方式

SpeedUP 은 여러분들의 스마트폰의 체감속도를 빠르게 조절 해줍니다.

바탕화면에 위젯을 올려 놓고 사용하시는 것이 가장 편리 합니다. 위젯을 올리는 법은 바탕화면을 꾸욱~ 길게 누르고 있으면 메뉴가 뜹니다. 그 중에 위젯을 선택하시고, 스피드업 위젯을 선택하시면 됩니다.

항상 시스템 관련 앱은 복잡하고 어려운 설정이 많아서 초보자들에게는 보기도 전에 사용을 못하는 경우를 많이 보면서 이 앱을 제작을 결심하게 되었습니다.
이 앱은 실행하고, 단 한번의 버튼 클릭으로 시스템을 관리 할 수 있게 해줍니다.

설치 후 실행을 하면 버튼이 5가지가 보입니다.

Go : 이 버튼을 누르면 현재 폰에서 가능한 가장 빠른 속도를 세팅해줍니다.
원래속도 : 폰의 기본 속도로 돌려놓아줍니다.
최고속도 : 가장 빠른 속도이기는 하지만 화면 전환의 에니메이션이 사라져 버리므로 미려함은 떨어지게 됩니다.
앱관리 : 캐시를 삭제 하거나 앱을 제거수 있는 창입니다.
나가기 : 앱을 종료하는 버튼입니다.

이 앱은 초보자를 타겟으로 제작된 것으로 설정이라던지 복잡한 사용법은 없습니다.
또한 앱을 종료 할 때는 현재 떠 있는 프로세스중에서 닫을 수 있는 것들은 닫아주고 종료되므로 메모리 관리에도 적당한 기능을 가지고 있습니다.

향후 스마트폰 최적화에 적당한 기능을 조금씩 추가하며 업그레이드 예정입니다.

자주 묻는 질문.

문 : 왜 마켓에는 업데이트가 되는데 스토어에는 업데이트가 아직 안되나요?
답 : 업데이트는 안드로이드 마켓과 각 스토어에 동시에 올립니다. 단지 각 스토어에서의 검증단계가 있기 때문에 등록되는 시점이 다릅니다. 또한 각 스토어에서 구매하신 앱은 각 스토어에서만 업데이트가 됩니다.

문 : 화면전환 속도만 가지고 팔아먹나요?
답 : 아닙니다. 설명을 잘 읽어봐주세요.. 현존하는 속도 관련 기능을 모두 포함하고 작동시키고 있습니다.

문 : 설치만 하면 되나요..?
답 : 설치 후 'Go' 버튼은 한번만 누르면 됩니다. 그 후에는 위젯을 바탕화면에 올려 놓으시고 필요시에만 그 위젯을 클릭해주시면 모든 기능이 자동으로 정리 됩니다.

문 : 종료 시키고 싶지 않은 앱은 어떻게 하나요?
답 : 스피드업을 실행 시키신 후 안드로이드 폰의 메뉴 키를 눌러서 '화이트 목록 관리'에 종료 시키고 싶지 않은 것을 체크해 주시면 됩니다.

문 : 슈퍼스피드 따라한 짝퉁이네?
답 : 스피드업이 오리지날 입니다. 스피드업을 따라한 짝퉁앱들이 많이 나왔지만 제대로 흉내 낸 앱은 없습니다. 유의해주세요.

高级任务管理器简称ATMGR，这是一款用来杀任务或进程的应用。它可以列出所有正在运行的任务或进程，你可以选择需要杀死的任务，点击“停止选中任务”来杀死进程。它是一款任务管理工具，可以对运行中的人物进行管理操作，你可以长按任务某个任务或者应用，将会弹出一个弹出菜单提供更多的操作选项。

市场里有很多的任务管理应用，排名最高的是Advanced Task Killer简称ATK的应用，我们的任务管理器和他们有一些区别。我们的任务管理器比ATK列出的任务更全，因为我们采用的和系统设置里列任务一样的算法，所以我们的结果更加准确。另一个终哟啊的不同是我们的任务管理器还具有卸载应用的功能，可以帮助你卸载没用的软件。所有的任务管理工具一半都有一个小部件，可以点击杀任务，我们的小部件有一个独特的功能，它可以动态显示当前的可用内存，不过这个功能不是默认开启的，需要在设置里启用。

安卓2.2系统改进了进程管理机制，导致了所有的任务管理器无法杀死服务或者任务栏图标。如果你想要彻底杀死一个任务，你可以长按住这个任务，然后选择“强制停止/详细信息”，你将会看到一个该任务详细信息的界面，点击“强制停止”即可。你也可以通过按“Menu”，点击“服务”，进入服务管理的界面，在这里你可以停止后台服务。

#常见问题
问：为什么有些应用杀了之后又会反复出现？
答：很多应用会被某些系统事件启动起来，任务管理工具没有办法阻止他们重启，但是可以停止他们。所以我们建议你们在设置里启用自动清理功能，它将在每次锁定屏幕的时候自动杀死这些任务，这将会延长电池使用时间，释放内存。
