I present to you the theme for the Bright Go Launcher EX.
This is a beautiful theme with bright light and transparency effects, with the original volume and the maximum large icons. After installing the theme all your applications change their icons and merged with the overall design.

-- new design for the menu for the latest version Go Launcher EX
-- 3 Background for icons
-- 3 Glass Effects
-- 370 additional volume icons as much as possible to go launcher ex resolution 85 * 85

-- 3 wallpaper

-- 2 App drawer background
-- Translucent backgrounds for the menu settings and folders
See the screenshots.

===>Important tips:
1. Make sure the latest version of GO Launcher EX has been installed! (Search “GO Launcher EX” to download)
2. Apply the Theme: MENU->Themes->Choose “Bright_theme” theme.


To change the icon size: MENU > Preferences > Character settings > Icons > Icon Size

How to change icons: Long press an icon > Replace > Theme's icons

How to change wallpaper: Press menu, then Wallpaper, then Go Wallpapers

Choice Dock background : MENU > Preferences > Visual settings > Backgroungs > Dock background > Go Theme background

Choice app drawer background : MENU > Preferences > Visual settings > Backgroungs > App drawer background > Go Theme background
