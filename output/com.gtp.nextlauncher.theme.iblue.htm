>> HOW TO INSTALL <<

1. Install the most recent GO Launcher EX for free from the android market.
2. On the home screen open the menu and tap themes, then select the cube theme.
3. To set a wallpaper open the home screen menu, tap wallpaper and then GO Launcher and take your pick!
4. If you have duplicated icons in the App Drawer, please clear the Cache and Defaults of GO Launcher EX, then open GO Launcher EX again and apply the theme. Please notice: this is not a problem with the theme, but GO Launcher EX.
5. If you have problems opening the app or selecting different icons please clean the cache for Go Launcher EX or try to reinstall the app again.

Important: The icon library when chosen a different icon it may take few seconds to load due to the size and the amount of icons in the icon library. Please give it a second. Thank you!
