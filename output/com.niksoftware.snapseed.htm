说明
Snapseed 是您唯一想每天使用的相片處理應用程式。您可輕鬆地給任何相片增添非凡特效，享受充滿樂趣的高品質相片體驗。

這款領先的數位相片處理軟體提供了眾多不可思議的先進功能，讓任何人都可以輕鬆增強、改變和分享其相片。內建 Google+ 功能使其更加強大，幫助您與親友分享您的圖像。

對 SNAPSEED 的讚譽：
「Snapseed…一流的相片編輯軟體。」~~Roy Furchgott，《紐約時報》

「Snapseed...對其效果和濾鏡處理具有令人難以置信的控制水平。」~~Jaymar Cabebe，CNET

優點：
• 使用「自動校正」，只需輕輕一點即可輕鬆調整您的相片
• 使用「調整圖像」，將您的相片調整至盡善盡美
• 使用「選擇性調整」，以增強相片中的特定物件或區域
• 可嘗試各種有趣的創意濾鏡，例如戲劇、復古、雜質以及移軸鏡攝影
• 為圖像增加邊框，以達到畫龍點睛的效果
• 透過 Google+、電子郵件和您安裝的任何其他服務分享您的作品。

支援：
• Android OS 4.0 (Ice Cream Sandwich) 或更高版本
• 電話或平板電腦
• JPEG 相片
• 支援語言：英文、德文、法文、西班牙文、義大利文、巴西葡萄牙文、日文、中文（繁體及簡體）以及阿拉伯文

聯絡方式：
google.com/+Snapseed

功能：
== 基本調整 ==
• 自動校正 - 自動分析相片並將色彩和曝光調整到最佳狀態。
• 選擇性調整 － 使用 Nik Software 革命性的 U Point 技術，在數秒鐘內對相片中的特定區域做出精準的選擇和增強。
• 調整圖像 － 使用「環境」來製造特別適合色彩和紋理的深度和自然飽和度。調整「白平衡」、「飽和度」和「對比度」等等。
• 拉直旋轉 － 使用簡單的手勢控制旋轉 90° 及/或拉直相片。
• 裁切 － 使用標準的縱橫比或自由裁切，輕鬆裁切圖像以去除相片上分散注意力的部分。

== 創新增強 ==
• 黑白 － 此濾鏡的靈感源於暗室，可為相片創造經典的黑白外觀。
• 復古膠片 － 使任何相片看起來像 50 年代、60 年代或 70 年代的古老彩色膠片照。
• 戲劇 － 透過為您的相片量身定製的效果來增添風格，從細微的紋理到異想天開的藝術效果都信手拈來。
• 雜質 － 使您的相片呈現完全獨特的時尚昏暗外觀。
• 移軸鏡攝影 － 建立窄聚焦帶，用以模擬微縮場景中常見的景深外觀。
• 細節 － 使用 Nik Software 專業產品具備的傳統銳化功能或獨特的細節構造控制來增強細節。
• 中心焦點 － 透過模糊和調節周圍背景的亮度來突出相片的拍攝主體。
• 相框 － 為相片增加風格化邊框，以達到畫龍點睛的完美效果。
• Retrolux － 讓您的相片帶有漏光、刮擦、膠片等多種風格，看上去充滿濃濃的復古風。
