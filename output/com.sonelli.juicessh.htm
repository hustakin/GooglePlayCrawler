安全隧道（SSH Tunnel），提供了一整套解决方案来帮助 Android 用户更快更安全的浏览互联网。本项目基于 GPLV3 发布，项目网站: http://sshtunnel.googlecode.com。

更新：由于账户部分逻辑已重新实现，因此在升级到 1.5.X 版本前，请卸载 1.4.X 及之前版本。

如果您在使用中遇到任何问题，请前往项目主页反馈，在市场中评论或留言并不能有效的解决您的问题。

注意事项：

1. 受限于手机性能，在使用本地SOCKS代理情况下，耗电量将大大增加，部分机型上还会遇到崩溃的情况。因此建议不要启用 “使用SOCKS代理”选项，并自行在服务器端部署HTTP(S)代理，或向服务商咨询远程代理的参数。

2. 请确认你的手机已 ROOT。在获得 ROOT 权限的前提下，安全隧道支持 WIFI, GPRS （CMNET）, EDGE 和 3G 等联网方式。并提供了设置全局代理，以及为指定程序设置代理的功能。

欢迎通过 Twitter 联系我们: @sshtunnel_dev

快速设置指南

使用本软件前请确认满足以下两个条件中的任意一个：1. 有自己的 VPS 或者独立服务器；2. 有专门服务商提供的 SSH 帐号。

服务器设置

满足第二项条件的用户可跳过此部分。

1. 在 VPS 或者服务器上安装任意 Linux 操作系统。

2. 安装 Squid 代理服务器。Ubuntu / Debian : sudo apt-get install squid, Redhat / CentOS / Fedora : sudo yum install squid

3. 以默认配置启动 Squid 即可。

手机设置

1. 下载 SSHTunnel 最新版

2. 启动后填入以下参数：

Host 服务器地址
Port 一般为 22，Puff 用户还可填 443
User, Password 你的 SSH 用户名与密码
Local Port 任意大于 1024 的数字即可，如 1984
PUFF 用户和自建服务用户，Remote Port 填入 3128
其他用户请选择“使用SOCKS代理”选项，Remote Port 留空即可

3. 点击 Connect 按钮

4. 非 Root 用户还需要手动设置手机的代理，如 127.0.0.1:1984
