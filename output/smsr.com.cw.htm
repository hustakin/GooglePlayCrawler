说明
这是 Google play 的头号应用，超过两百万次下载。
倒计时小工具 是提醒到最重要的事件的一种免费的应用，很方便使用。
开始用这种应用以后，千万不要错过生日，纪念日，婚礼或游戏。你是用自己的手指为了倒计时 到最重要的事件（交货，兵役，薪金，奖金，假期，圣诞节，新年或滑雪）还有多少时间？你有没有遗忘过一些重要的日子， 然后然后感到了不好意思？有了这种部件的帮助下，将不会错过任何重要事件。
有了一个新的，美丽的高解析度的主题，这种 部件不仅是有用的，但看起来 也 很漂亮。因为重要的事件应该根朋友们分享， 你也能把他们通过 Facebook, Twitter, Google+ 或电子邮件分享。
你也能加各种各样的几个事件：设定受到工作上的奖金的日期， 第二天到夏威夷的旅行。 到了那个日期的时候，部件提前开始倒计时， 那么您就知道您是什么时候停止吸烟或已经有多少时间您节食。
倒计时小工具 （倒计时部件）有三种不同的尺寸：1x1, 2x1 和3x1。 小的部件和中的部件您能使用为了倒计时天数。大的部件您也能使用为了倒计时小时和分钟。
部件的特性：
小的部件 （1x1）简单的图形，显示事件的天数。
中的部件 （2x1）适度的图形，显示到事件的天数 （到四位数）。
大的部件 （3x1）丰富的高清图形，显示到事件的天数，小时和分钟。

怎么用? 很容易！ 您开始用部件之后，输入事件的日期和名字。

Adding Widgets for Android 3.x+ (Honeycomb, ICS, Jelly Bean)
------------------------------------------------------------
1. Touch the All Apps icon on your Homescreen.
2. Touch Widgets tab at the top of the screen.
3. From the main Widgets screen, you can
4. Move between screens to find Final Countdown: Swipe left or right.
5. Drag a countdown widget to a Home screen: Touch and hold the widget's icon, slide your finger where you want to place it, and lift your finger.
6. Choose an event from Events list to activate widget.

Adding Widgets for Android 1.x, 2.x (Eclair, Gingerbread, Froyo)
----------------------------------------------------
1. Long-click on an empty space of your homescreen.
2. Select Widgets from Add to Home screen.
3. Select Final Countdown.
4. Select an event from Events list.

Deleting Events
----------------------------
You can delete an event by going to Events tab and click the Edit button (pencil icon) on the top right corner. Then simply click on the Trash icon next to the event you want to delete.

Check FAQ on app's website for problem resolutions.


**Notes**
-------------
If widgets aren't listed, please make sure that the application is installed to internal storage.

Events that appear to have one hour off are due to DST.

Final Countdown uses the local timezone of the device it is running on. This can be changed in the OS Settings.

Task Killer/Task Manager apps can cause countdown widgets to freeze. If you have any of those installed, configure them to make an exception for Final Countdown.


**TIPS**

- Add a new widget to jump start stopped widgets.
