说明
The ANT Radio Service enables access to ANT wireless communication on your device*. ANT is an extremely power efficient wireless communications technology. When this service is installed it cannot be launched directly, but will be used by other applications to use ANT wireless communications, including ANT+™ enabled applications.
*[use of ANT technology requires a supported phone or an adapter, see hardware requirements below]


One of the largest uses of ANT is the ANT+ network of devices. ANT+ is an established set of defined communication profiles which allows communication with many of the top sport, fitness and personal health applications such as a heart rate straps, running foot pods, cycling power meters, weight scales, blood pressure monitors and more. ANT+ enabled applications typically also require downloading the ANT+ Plugins http://play.google.com/store/apps/details?id=com.dsi.ant.plugins.antplus.

HARDWARE REQUIREMENTS
For detailed requirements visit http://www.thisisant.com/consumer/ant-101/ant-in-phones/
You will need one of the following hardware configurations to connect ANT+ devices:


1. A device with native ANT support. Many models of phones include built-in ANT technology including models from Samsung, HTC, Motorola, and Sony/Sony Ericsson (for an official list of models visit http://www.thisisant.com/consumer/ant-101/ant-in-phones/).
2. An Android device with the USB Host feature (optional in Android 3.1 and up) and an ANT USB Stick (ANTUSB2 or ANTUSB-m) through a USB OTG cable or host A port. Requires the ANT USB Service.
3. An Android device with the USB Accessory feature (optional in Android 2.3.4 and up) and an ANT Android USB Accessory, such as a SelfLoops ANT adapter.


FAQ:
Why is this pre-installed on my phone?
This service is pre-installed as a system application on phones that provide built-in ANT wireless support to help enable the full feature set of your phone. This is comparable to other system applications such as WiFi, Bluetooth, Camera, Contacts, Phone, etc. except with the added benefit of it being updatable to the latest version without a firmware update. In the pre-installed scenario, Android will not let you uninstall the service as it is considered a "system app”. However, even if you don't plan on using ANT, you should know that the ANT Radio Service takes up very little space and will only run if it is invoked by an application.


Why does this need the ‘Network communication- Full Internet access’ permission?
The ANT Radio Service will never download or upload any data over the internet. This permission is only required because it is possible to install the ANT Radio Service on an Android Emulator (ie: for development purposes) in which case it requires access to internet sockets to communicate with an ANT USB Stick on the host Windows PC using the ANT Android Emulator Tools. This communication requires the INTERNET permission but only communicates with the local PC.


Why does this need the ‘System tools- Modify global system settings’ permission?
This allows initialization of whether ANT should be turned off or kept on when Airplane mode is entered.

常见问题：
这是为什么我的手机上预装？
这项服务是作为一个系统应用程序提供内置ANT无线支持，以帮助您的手机的全部功能的手机上预装。这是与其他系统应用程式，如WiFi，蓝牙，摄像头，联系人，联系电话等，除非它是没有固件更新到最新版本更新的另外一个好处。在预装的情况下，Android将不会让您卸载该服务，因为它被认为是一个“系统应用”，但是，即使你不打算使用ANT，你应该知道，ANT无线服务占用很少空间和只会运行，如果它是由一个应用程序调用。


为什么会出现这种需要“网络通信完整的Internet访问权限？
   从来没有的ANT无线服务将下载或上传任何数据在互联网上。此权限仅必需的，因为它可以安装Android模拟器的ANT无线服务（即：发展的目的），在这种情况下，它需要访问互联网插座沟通与主机上的Windows PC上使用的ANT ANT USB记忆棒Android模拟器工具。这种通信需要INTERNET权限，但只与本地PC。


为什么会出现这种需要“系统工具修改全局系统设置的权限？
   这允许初始化蚂蚁是否应该关闭或保存时进入飞航模式。

Limitations.

The free version of IpBike is limited to one million wheel revolutions (or equivalent in gps only mode 2070mm wheel). For permanent use after this you will need to get the IpBikeKey app which will unlock this limitation.

IpBike uses IpSensorMan to provide access to ANT+, Btle and BlueTooth sensors. If you have a compatible phone and do not have IpSensorMan installed you will be prompted to install it. You can use an ANT USB Stick if your phone supports USB host mode.
