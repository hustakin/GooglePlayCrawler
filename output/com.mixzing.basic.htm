Poweramp v2.x 的常见问答：
问：我的歌曲从文件夹/音乐库中消失了。
答：请确保您已将所有音乐文件夹关联到“Poweramp 设置 => 文件夹/媒体库 => 音乐文件夹”。
您的初始安卓音乐库未更改，亦未删除任何文件。

问：冻结/文件夹/音乐库扫描过慢
答：您的 SD 卡（半）损坏。 请在电脑上检查您的 SD 卡错误或尝试使用其他 SD 卡。
