说明
The only iPhone Theme for Go Launcher EX with icons the same size as the iOS Retina Edition. This theme also included the largest library of icons with pre-customized icons from actual market applications!!!!

iOS Retina HD is a theme for GO Launcher EX, inspired by the iPhone's iOS Retina Edition. iOS Retina HD was carefully created to simulate and improve the GUI interface of the iPhone, including the buttons, the Tab icons, pre-made iOS icons , folders and Tab Menu as well as icons for select Market Apps.
More icons to come in new upgrades

What is included in iOS Retina HD?:

- Icons of 116px x 116px that can be scale to their maximum size using Go Launcher EX icons size option.
- 411 icons iOS 4 style to replace your system apps and Market apps.
- Tab Bar, Tab icon & Buttons with the same look of iPhone.
- Background of the App Drawer and folders just like iOS.
- Icon & Text Shadows on the main screen.
- Tons of news icons to come with the next upgrade!

>> HOW TO INSTALL <<

1. Install the most recent GO Launcher EX for free from the android market.
2. On the home screen open the menu and tap themes, then select the cube theme.
3. To set a wallpaper open the home screen menu, tap wallpaper and then GO Launcher and take your pick!
4. If you have duplicated icons in the App Drawer, please clear the Cache and Defaults of GO Launcher EX, then open GO Launcher EX again and apply the theme. Please notice: this is not a problem with the theme, but GO Launcher EX.
5. If you have problems opening the app or selecting different icons please clean the cache for Go Launcher EX or try to reinstall the app again.

Important: The icon library when chosen a different icon it may take few seconds to load due to the size and the amount of icons in the icon library. Please give it a second. Thank you!
