说明
您智慧型手機中成堆的相片可能雜亂無章。別讓您的珍貴回憶在手機某個被遺忘的角落褪去光彩。有了 million moments，一個可以檢視與分享照片的應用程式，讓您再次喚醒回憶。且在如同雜誌般雅致的版面中，您可以欣賞並分享您最愛的相片。
拍照、匯入與瀏覽相片。
讓您優雅又輕鬆地探訪您的回憶。

= = = = = = = = = = = = = = = = =
million moments 在 Google Play 攝影類別中，在日本、印度與新加坡曾經列為熱門免費應用程式。
= = = = = = = = = = = = = = = = =

總覽：
－不必編輯相片，直接建立相簿。
從您的智慧型手機或 Facebook 相簿中匯入相片，或者從 million moments 中拍攝新相片，並自動建立相簿。匯入的相片會自動依日期排序，每頁顯示 1 到 5 張相片。
－附加標籤以建立新相簿
在相片上附加標籤會自動建立和標籤對應的新相簿。例如，建立以子女名稱命名的標籤，接著將其附加在子女的相片上，您便能擁有以您子女命名的相簿，其中包含子女的相片。
－內建輕鬆分享相片功能
million moments 可以在 Facebook 上張貼相簿的專頁。

常見問題：
－可以將版面變更為顯示其他部分的相片嗎？
可以。請按住相片、放開、然後拖曳相片。
－如何刪除相片？
輕按您想刪除的相片，然後輕按底部的垃圾桶圖示。

・アルバムのお気に入りページを、美しいレイアウトのままFacebookへ投稿、共有できます。
