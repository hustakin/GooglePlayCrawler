【Sleipnir Mobile Black Edition について】

Sleipnir Mobile（無料）の機能に「検索エンジン選択」の搭載と、黒を基調としたアイコンと画面デザインを採用した有料アプリケーションです。

デザインの変更箇所は以下の3箇所です。
• アプリケーションアイコン
• アドレスバー
• 新規タブ追加時に表示される「いつものサイト」

※ 通常版から乗り換えの際は、バックアップ機能でタブ、ブックマーク、設定を移行することができます。
※ タブバーの色はタブを上にフリックすることで変更できます
※ 検索サジェスト機能は、検索エンジンが Yahoo! JAPAN に設定されているときにのみご利用いただけます。

これらの機能が必要ない場合は、無料の Sleipnir Mobile をご利用ください。
https://play.google.com/store/apps/details?id=jp.co.fenrir.android.sleipnir


【3つの魅力】

★ 速さと精度のために1ミリ単位で調整されたジェスチャ

★ 「開く」「閉じる」「切り替える」の素早さを極めたタブ

★ いつもの操作が1/3になるカスタマイズ性

【特徴的な機能】

• ジェスチャ
タブ切り替え、更新、ブックマーク表示、全画面化など多くの操作が、たったの1ステップで完了します。
　
• Hold And Go
あとで読みたいリンクは長押しするだけで、隣にタブを開いて読み込ませておくことができます。

• カスタムボタン／カスタムパネル
柔軟にカスタマイズできるUIで、お好きな機能・設定・ブックマーク（ブックマークレット）を1ステップで届くところに配置できます。

• オフライン表示
開いているタブの内容は自動的に保存されるので、一旦終了しても保存データから一瞬のうちに復元します。だから電波を気にせずいつでもどこでもすぐに確認できます。（※ Android 3.0 以上）

• タブのメモリを自動解放
たくさんタブを開いても自動的に古いタブのメモリを解放するので、100タブあっても軽くて安定しています。解放されたタブは次回表示するときにすぐ復元します。

• スクロールグリップ
どんなに長いページでも、スクロールバーをタッチして目的の位置まで一気にスクロールできます。

• タブ・フリック
タブを下にフリックするだけでそのタブを閉じることができます。また上にフリックでグループ管理、上から右にフリックで新規タブを開くこともできます。

• 人気トピックス
ニュースや人気記事、人気サイトの最新記事をまとめてチェックすることができます。

• 表示モード
指定したページやサイトを開いたとき、自動的にユーザーエージェント(User Agent)を切り替えたり別のアプリで開くことができます。（※アドレスバー長押しメニューより）

• ピックアップ検索
表示中のページや別アプリから受け取ったテキスト内のキーワードをタップするだけで検索することができます。（※メニューボタンもしくはアドレスバー長押しメニューより）

• エクステンション
Web ページの見た目を変更したり、特定の機能を追加することができます。エクステンションのダウンロードは、Extensions Gallery ( http://extensions.fenrir-inc.com/ )から。エクステンションのインストールは、インストールボタンをタップするだけで完了します。

• Fenrir Pass（フェンリル・パス）
Windows や Mac、iPhone、Windows Phone 上の Sleipnir と自動でブックマーク同期や、統合された様々なウェブサービスを利用できる無料のクラウドサービスです。
対応サービス: Twitter / Facebook / Google+ / LinkedIn / Tumblr / Flickr / Readability / Instapaper / Pocket / Dropbox / Evernote / SkyDrive / Google Reader

• 設定不要の広告ブロック（※ Android 3.0 以上）

ドルフィンブラウザ（Dolphin Browser）は、最速のブラウジングと使いやすさを誇るAndroidブラウザです。


拡張機能を使って、パソコンのブックマークを同期。Facebook（フェイスブック）、Twitter（ツイッター）、Evernote（エバーノート）への共有も楽々。ジェスチャや音声で片手ブラウジング。タブブラウザなので、ページの切り替えも簡単。フラッシュコンテンツにも対応。アドオンによるカスタマイズで、自分だけのブラウザに。世界で8000万人が使う最速で使いやすいブラウジングを体験してみませんか。


ブックマークをパソコンと同期！
ドルフィンの拡張機能（エクステンション）*を使えば、ワンタップでパソコンのChrome（クローム）またはFirefox（ファイヤーフォックス）とブックマークを同期できます。ブックマークは左端から右にスワイプするだけで、サイドバーから簡単にアクセスできます。


ブラウザからの共有が簡単！
ウェブページをドルフィンから直接 Facebook や Twitter に投稿したり、Evernote や Box に保存することができます。特に、Evernote にここまで簡単にウェブクリッピングができるのは、ドルフィンだけ！共有するために一度ブラウザーを離れる必要がなく、投稿後は引き続きブラウジングができるから共有がより手軽に。


パソコンのようなブラウジング体験
タブブラウジングなので、面倒な画面の移動がありません。ウェブページを素早く簡単に開いたり、切り替えたりできます。一度閉じたタブを復元することも簡単。ユーザエージェントを使ってAndroid端末でもパソコン（デスクトップモード）やiPhone画面に切り替えられます。サイドバーによるブックマークの呼び出しや管理のしやすさも、パソコンのブラウザー並み。


デスクトップとモバイルで情報を送受信
ドルフィンの拡張機能*を使えば、パソコンのChrome、Firefox、Safari（サファリ）とウェブページ、地図、電話番号などを簡単に共有できます。


端末を複数持っている人におすすめ！
FacebookまたはGoogleアカウントを使ってドルフィンコネクトに登録をすれば、Android、iOS、デスクトップ間で履歴、ブックマーク、パスワード、開いているタブまで同期することができます。


片手でも楽々ブラウジング
音声認識のソナーや、自分だけのサインが作れるジェスチャを使えば、ウェブ検索だけでなく、新規タブを開く、ブラウザーの終了、ブックマークなどのブラウジング操作も音声や指一本で行えます。


Webゲームを速く、楽しく
HTML5対応のドルフィンブラウザ。Webゲームを楽しむ方にはその速さをより実感していただけます。デバイスにFlash Playerをインストールし、ドルフィンの設定からプラグインを有効にすることでフラッシュコンテンツもご覧いただけます。


自分好みのブラウザにカスタマイズ
明るさコントロールやスクリーンカットなど、60以上のアドオンの中からニーズに合ったアドオンをインストールすれば、更に便利なブラウジングを体験していただけます。アドオンへのアクセスは、右端から左にスワイプするだけ！スタートページにはよくアクセスするサイトを登録できるスピードダイヤルがあるので、ワンタッチでお気に入りのモバイルサイトにアクセスできます。新たに追加されたテーマ機能で、ブラウザ画面をお気に入りの色や背景にカスタマイズすることもできます。


プライバシーを重視した、安心、安全なブラウジング
メニュー > 設定からあらかじめ設定できることはもちろん、ブラウザの終了時にもキャッシュや履歴をクリアするポップアップが表示されるため、簡単にプライバシーコントロールをしていただくことができます。
