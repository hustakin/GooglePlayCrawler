การ Update ข้อมูล
- ข้อมูลละครและรายการทีวีจะทำการ Update เป็นระยะๆ
- การเชื่อมต่อแบบ Wi-Fi วิดีโอและสัญญาณภาพที่ได้จะมีความคมชัดกว่า 3G และ EDGE
- กรุณาตรวจสอบปริมาณการใช้ข้อมูลดาต้าในแพคเก็จมือถือของท่านอยู่เสมอ (Data Usage) เพื่อไม่ให้เกิดค่าใช้จ่ายที่เกินจำเป็น


คำแนะนำการใช้งานเบื้องต้น
สำหรับการดูทีวีหรือวิดีโอจำเป็นต้องมีโปรแกรม Flash Player ให้ทำตามวิธีต่อไปนี้
1. ออกจากโปรแกรม
2. ที่หน้าหลักของมือถือให้เข้าไปในเมนู Settings (การตั้งค่า) => Security (ความปลอดภัย) => เช็คถูกที่ "Unknow Source" (ไม่ทราบแหล่งที่มา) แล้วกด OK (ตกลง)
3. เข้าเว็บ http://tinyurl.com/flash4android จากบราวเซอร์บนมือถือ
4. ระบบจะดาวน์โหลดไฟล์ flash11.apk อัตโนมัติ
5. ติดตั้งไฟล์ flash11.apk ที่ดาวน์โหลดจากเว็บข้างต้น
6. ลองปิดแล้วเปิดโปรแกรม Thai TV+ เพิ่อทดสอบหรือปิดเปิดเครื่องใหม่
