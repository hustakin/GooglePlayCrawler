(자세한 내용은 본 설명문 또는 네이버에서 "천지인 더블 휴대폰 설치 방법"을 검색하세요)
3. "천지인 더블 키보드" 애플리케이션에는 "천지인", "천지인 더블", "쿼티", "쿼티 더블" 4개 방식이 포함되어 있습니다. ( 키보드 설정은 한/영 자판 1초정도 "꾹" 누르면 키보드 설정에서 선택할 수 있습니다. )

[ "천지인 더블 키보드" 애플리케이션 설치 및 사용방법 ]
-> 설치 및 사용방법은 다른 한글 입력기와 동일합니다.
-> 중간에 나타나는 팝업 문구는 일반적인 사항이니 안심하셔도 됩니다.

첫째, "천지인 더블 키보드" 애플리케이션을 다운로드 합니다.
둘째,
핸드폰 환경설정(톱니모양 아이콘) -> 메뉴중 "언어및키보드
또는 언어및입력 터치 -> "천지인 더블 키보드"에 체크
셋째,
메모장,메시지,카카오톡 등 입력창 -> 텍스트 액정 화면을 1~2초간 "꾹" 누르기(진저 브레드 버전) 또는 휴대폰 상단의 키보드 모양의 아이콘을 아래로 드래그하기(아이스크림 샌드위치 버전) -> 입력방식 선택 -> "천지인 더블 키보드"로 변경 체크 하면 사용할 수 있습니다.
. 팬택 베가 기종의 경우 한글 키패드 좌측 하단에 있는 "1 별표" 키패드를 꾹 눌러서 바꿔야 할 수도 있다.

2. 쿼티 더블 키보드를 이용하면 훨씬 편리합니다.
가 모음 ( ㅐ ) ( ㅔ ) 가 없고 대신 그 위치에 모음 ( ㅘ ) ( ㅝ ) 가 있습니다.
나 모음 입력시 "왔다 갔다" 하지 않고 "제자리"에서 더블 터치만 하면 오타가 나지 않습니다.

== 한자 한글 키보드 프로는 절대!!! 개인정보를 수집하지 않습니다. ==
설치시 뜨는 개인정보 수집 메세지는 구글에서 사고예방 차원으로 띄우는 문구로서, 모든 키보드 설치시 뜨게 됩니다. 한자 한글 키보드 프로는 어떠한 방법으로도 개인정보를 수집하거나 요구하지 않습니다.

구매를 취소하시는 분들께서는 번거로우시더라도 fancyydk@gmail.com으로 개발자에게 취소 이유를 보내주시면 감사하겠습니다. 소중한 의견을 추후 업데이트에 반영할 수 있도록 하겠습니다.

==설치 방법==
1.마켓에서 다운로드를 받으신 후
2.환경설정 -> 언어 및 키보드 -> Sino Korean Keyboard Pro를 체크합니다.
3.주의 메시지는 원래 키보드 설치시 뜨는 메시지이므로 확인하시면 됩니다.
4.카카오톡 등 글자 입력창을 꾹 누르면 입력 방법 메뉴가 뜨는데 거기서 Sino Korean Keyboard Pro를 선택합니다.
*체크만 하면 키보드가 변경되지 않습니다. 반드시 "입력 방법"에서 Sino Korean Keyboard Pro를 선택하셔야 변경됩니다.

==특징==
1.한글 글자에 해당하는 한자를 한번에 다 보여주고 선택할 수 있게 했습니다.
2.QWERTY(두벌식) 키보드 외에 천지인과 단모음을 추가하였습니다.
3.사용자가 키보드 키의 높이를 3단계로 조절할 수 있습니다.
3.스페이스바 양 옆에 방향키를 추가해서 글 입력시 수정이 쉽도록 했습니다.
4.특수기호중 많이 사용하는 키를 앞쪽에 배치하였습니다.
5.한자 외에도 윈도우PC에서처럼 한글 자음을 입력하고 한자키를 누르면 특수문자를 입력할 수 있게 했습니다.
