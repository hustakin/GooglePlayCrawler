[제품소개]
"알송 안드로이드"는 실시간 싱크가사를 제공하는 모바일 음악 플레이어입니다.

[주요 기능]
1. 싱크 가사 제공
- 3G/Wi-Fi 연결을 통해 재생 중인 노래의 가사를 자동으로 연결합니다. 캐시로 저장되어 이후에는 데이터 접속 없이도 볼 수 있습니다.
2. 자동 태그 가사 보기
- 서버가사 또는 IRC 가사를 보여줄 수 없을 때, 태그에 가사가 있는 경우 자동으로 태그 가사를 보여줍니다. (환경설정-고급 설정-자동 태그 가사 보기를 ON으로 설정할 경우)
3. 재생목록 만들기
- 사용자가 재생목록을 만들어 곡 목록을 관리할 수 있습니다. 재생목록 메뉴에서 ‘새 재생목록 추가’를 누르면 재생목록을 만들 수 있습니다.
4. 재생목록 별 음악감상
- 여러 개의 재생목록을 생성하여 재생목록 별로 음악을 감상할 수 있습니다.
5. 트위터 공유
- 재생화면 상단의 화살표 버튼을 누르면 재생중인 곡의 정보를 트위터로 공유할 수 있습니다.
6. 위젯 지원
-앱을 실행시키지 않고도 위젯을 통해 알송을 사용할 수 있습니다. (2X2, 4X1 사이즈 지원)
7. 타이머 기능
- 알송이 재생될 시간을 설정하여 설정된 시간이 지나면 자동으로 알송이 종료됩니다. (환경설정-고급 설정-타이머 설정)
8. 구간 반복 기능
- 재생화면 상단의 구간 반복 버튼을 눌러 시작위치와 종료위치를 설정한 후 구간 반복 기능을 사용할 수 있습니다.
9. 가사 검색 기능
- 검색하고자 하는 가사가 있을 때 재생화면 상단의 가사 검색 버튼(돋보기 모양)을 눌러 직접 가사를 적용할 수 있습니다.
10. 벅스 뮤직서비스 제공
- 벅스뮤직에서 제공하는 온라인 스트리밍 음원을 언제 어디서나 즐길 수 있습니다.

[사용 환경]
안드로이드 2.2, 2.3, 4.0, 4,1, 4,2 지원

※ 여러 부가적 기능들이 추후 업데이트 될 예정입니다.

[V2.5.8 업데이트 내역]
1. 플레이리스트 담기 최대 1,000곡으로 확대
플레이리스트에 담을 수 있는 최대 곡 수가 500곡에서 1,000곡으로 늘어났습니다. 이제 플레이리스트에 더욱 많은 곡들을 한번에 담아 쉽고 편하게 음악을 감상해 보세요.
단, 저사양 기기에서는 천 곡을 한번에 담거나, 곡 정렬시에 화면이 잠시 멈추는 현상이 발생할 수 있습니다. 이 경우 플레이리스트에 담은 곡 수를 천 곡보다 낮은 수로 유지해주시면 쾌적하게 서비스를 이용하실 수 있습니다.

2. 플레이리스트 곡 검색해서 바로 듣거나 삭제하기 기능 추가
플레이리스트에 담은 곡 중, 내가 원하는 곡을 쉽고 빠르게 검색하여 음악을 듣거나, 플레이리스트에서 곡을 삭제할 수 있습니다. 또한, 플레이리스트상에서 해당 곡이 위치한 곳으로 빠르게 이동할 수 있습니다. 지금 바로 플레이리스트 화면 위에 추가된 검색바를 눌러 원하는 검색어를 입력해 보세요(초성검색도 지원한답니다. ex – ‘ㄱㅇㅂ’을 입력하면 ‘가을밤’을 찾을 수 있습니다).

3. 플레이리스트 중복곡 삭제 기능 추가
플레이리스트에 동일한 곡이 여러 번 들어가 있는 경우, 한 곡만 남겨두고 나머지는 삭제할 수 있는 기능입니다. 플레이리스트 위 쪽에 있는 [중복곡 삭제] 버튼을 눌러 보세요.
단, 곡의 유형이 다른 경우 서로 다른 곡으로 인식합니다(ex. 동일한 곡이 MP3와 스마트폰 전용 다운로드 음원의 두 가지 형태로 존재할 경우, 서로 다른 곡으로 인식해 중복곡 삭제가 작동하지 않습니다).

4. 플레이리스트 및 마이앨범 담기 위치 설정 옵션 추가
플레이리스트에 곡을 담을 때, 목록의 맨 처음에 담을 수 있는 옵션이 추가되었습니다. 또한, 마이앨범에 곡을 담을 때 목록의 맨 처음뿐만 아니라 맨 마지막에 담을 수 있도록 설정 옵션이 추가되었습니다.
[설정 > 플레이리스트/마이앨범 담기 설정 > 담기 위치 설정]에서 설정할 수 있습니다.

5. 가사보기 위치 고정 기능 추가
플레이어에서 가사를 볼 때, 화면을 스크롤하면 해당 위치에 화면이 고정돼 있도록 기능 추가되었습니다. 다시 원래 위치대로 가사가 자동 스크롤되게 하고 싶으시면 화면 오른쪽의 돋보기 버튼 위에 있는 [현재 위치로] 버튼을 누르시면 이전 상태로 돌아갑니다.
